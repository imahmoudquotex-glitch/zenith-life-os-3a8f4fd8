## 🧠 Master Architect Upgrade — Focus Session Reliability Gate

- **Server-authoritative time:** لا تقبل client delta. heartbeat يحسب server-side، clamp، crash recovery، وidempotent session events.
- **Audio lifecycle:** كل AudioContext/sources/promises تتنضف عند pause/end/unmount/navigation، مع tests تمنع memory/audio leaks.
- **Free sound policy:** كل soundscapes الأساسية والرفع الشخصي ضمن الحدود مجانية، و`is_premium` يبقى false أو يُحذف من schema لتجنب drift.
- **Anonymous co-working:** rooms تعرض counts فقط، no names, no rankings, no minutes comparisons. privacy by default.
- **Accessibility:** timer announcements screen-reader friendly، keyboard-only controls، reduced motion، وvisual focus قوي في dark mode.

### Focus Baseline

- Server يحسب الـ delta من `last_heartbeat_at` ولا يقبل client-supplied focused delta.
- Clamp بـ `LEAST(server_delta, 10s)` لمنع over-counting لو tab كان suspended.
- لو الـ delta أكبر من 60s تعتبر الجلسة crashed ولا يُحسب الوقت الزائد.
- Audio cleanup كامل: cancel pending promises + stop all sources + close context.
- كل الـ 8 soundscapes الأساسية مجانية.
- Custom audio upload إلى R2 مجاني: 50MB حد أقصى per session، و500MB إجمالي per user.
- لا يوجد premium sounds.
- Co-working rooms anonymous فقط: "3 أشخاص بيذاكروا الآن" بدون أسماء وبدون leaderboard.
- Custom audio storage على Cloudflare R2.

### Heartbeat RPC (Server-Authoritative)

```sql
-- Migration 0254b — server-authoritative heartbeat
CREATE OR REPLACE FUNCTION focus_heartbeat(p_session uuid)
RETURNS table(actual_seconds int, status text)
LANGUAGE plpgsql SECURITY DEFINER AS $$
DECLARE
  v_delta int;
BEGIN
  -- Compute server-side delta, clamp to 10s max
  UPDATE focus_sessions_v2 SET
    actual_focused_seconds = actual_focused_seconds +
      LEAST(EXTRACT(EPOCH FROM (now() - last_heartbeat_at))::int, 10),
    last_heartbeat_at = now()
  WHERE id = p_session
    AND status = 'running'
    AND user_id = auth.uid()
  RETURNING actual_focused_seconds, status INTO actual_seconds, status;

  RETURN NEXT;
END;
$$;

REVOKE EXECUTE ON FUNCTION focus_heartbeat FROM PUBLIC;
GRANT EXECUTE ON FUNCTION focus_heartbeat TO authenticated;
```

### AudioContext Cleanup

```tsx
// /src/lib/focus/audio-mixer.ts
export function useFocusAudio(tracks: Track[]) {
  useEffect(() => {
    let cancelled = false
    const ctx = new AudioContext()
    const sources: AudioBufferSourceNode[] = []
    const gains: GainNode[] = []

    Promise.all(tracks.map(async (t) => {
      const buffer = await fetchAudioBuffer(t.url, ctx)
      if (cancelled) return  // Don't mutate after unmount
      const src = ctx.createBufferSource()
      const gain = ctx.createGain()
      gain.gain.value = t.volume
      src.buffer = buffer
      src.loop = true
      src.connect(gain).connect(ctx.destination)
      src.start()
      sources.push(src)
      gains.push(gain)
    })).catch(() => { /* aborted */ })

    return () => {
      cancelled = true
      sources.forEach((s) => { try { s.stop() } catch {} })
      gains.forEach((g) => g.disconnect())
      ctx.close().catch(() => {})
    }
  }, [tracks])
}
```

## 🎯 نظرة عامة

مش مجرد Pomodoro عادي. ده نظام تركيز كامل: المستخدم بيختار **عادة محددة** (مذاكرة إنجليزي، كتابة كود، قراءة، رياضة، …) ويعدّل **target يومي بالساعات** (مثلاً 5 ساعات إنجليزي/يوم). لما يضغط "ابدأ جلسة تركيز":

1. يختار technique: **Pomodoro 25/5** أو **Deep Work 50/10** أو **Ultradian 90/20** أو **Custom**.
2. يختار soundscape (مطر/رعد/كافيه/غابة/أمواج/Brown Noise/Binaural Beats 40Hz) أو **يرفع ملف صوتي خاص بيه**.
3. الجلسة تبدأ — لو وقف فجأة (closed tab / crash / pause) **الوقت اللي اتسجل لحد لحظة الوقف بيتسجل تلقائياً**.
4. بعد كل work block يطلع break صغير مع صوت لطيف، وفي نهاية اليوم يحدّث الـ progress للعادة.

ده القلب الحقيقي للـ LifeOS — العادة + التركيز + الموسيقى في حلقة واحدة.

### 2) محرك الجلسة (Session Engine) + Resilience

```tsx
// /src/services/focus/engine.ts
export async function startSession(input: {
  userId: string
  habitId?: string
  technique: 'pomodoro_25_5' | 'deep_50_10' | 'ultradian_90_20' | 'custom'
  customWork?: number
  customBreak?: number
  plannedBlocks: number
  soundscapeId?: string
  customAudioUrl?: string
  moodBefore?: number
}) {
  const presets = {
    pomodoro_25_5:  { work: 25, brk: 5 },
    deep_50_10:     { work: 50, brk: 10 },
    ultradian_90_20:{ work: 90, brk: 20 },
    custom:         { work: input.customWork ?? 30, brk: input.customBreak ?? 5 }
  }
  const p = presets[input.technique]
  const session = await db.from('focus_sessions_v2').insert({
    user_id: input.userId,
    habit_id: input.habitId,
    technique: input.technique,
    work_block_minutes: p.work,
    break_block_minutes: p.brk,
    planned_blocks: input.plannedBlocks,
    soundscape_id: input.soundscapeId,
    custom_audio_url: input.customAudioUrl,
    mood_before: input.moodBefore,
    status: 'running'
  }).select().single()
  await logEvent(session.data.id, 'start', { technique: input.technique })
  return session.data
}
```

### 3) Client Timer + Service Worker (background safe)

```tsx
// /src/lib/focus/timer.ts (runs in Service Worker so it survives tab blur)
self.addEventListener('message', (e) => {
  if (e.data.type === 'START') startTimer(e.data.sessionId, e.data.workMin)
})
function startTimer(sessionId: string, workMin: number) {
  const endAt = Date.now() + workMin * 60_000
  const tick = setInterval(async () => {
    const remaining = Math.max(0, endAt - Date.now())
    // Post heartbeat every 5s
    await fetch('/api/focus/heartbeat', { method: 'POST', body: JSON.stringify({ sessionId }) })
    // Notify clients of remaining time
    self.clients.matchAll().then(cs => cs.forEach(c => c.postMessage({ type: 'TICK', remaining })))
    if (remaining <= 0) { clearInterval(tick); finishBlock(sessionId) }
  }, 5_000)
}
```

### 4) Audio Engine — multi-track + custom upload

```tsx
// /src/components/focus/AudioMixer.tsx
import { useEffect, useRef } from 'react'
export function AudioMixer({ tracks }: { tracks: Array<{ url: string; volume: number }> }) {
  const ctxRef = useRef<AudioContext>()
  useEffect(() => {
    const ctx = new AudioContext()
    ctxRef.current = ctx
    const sources = tracks.map(async t => {
      const buf = await fetch(t.url).then(r => r.arrayBuffer())
      const decoded = await ctx.decodeAudioData(buf)
      const src = ctx.createBufferSource()
      src.buffer = decoded; src.loop = true
      const gain = ctx.createGain(); gain.gain.value = t.volume
      src.connect(gain).connect(ctx.destination); src.start()
      return { src, gain }
    })
    return () => { ctx.close() }
  }, [tracks])
  return null
}
```

- Custom upload: drag & drop ملف mp3/wav → upload to Cloudflare R2 → URL محفوظ في `user_soundpacks` أو `focus_sessions_v2.custom_audio_url`

- Format validation server-side
- Auto-fade on session end (2s fade-out)
- Background playback يكمل لو الـ tab راح background (Service Worker + Media Session API)

### 5) Crash Recovery + Auto-Save

الأهم: لو المستخدم قفل التاب فجأة، crashed جهاز، أو نام —

- آخر heartbeat هو لحظة الحفظ
- لما يرجع يفتح الموقع: لو فيه session بحالة `running` بس آخر heartbeat من أكتر من 30 ثانية → modal: "ارجعت! سجلنا لك X دقيقة. تكمّل ولا تقفل الجلسة؟"
- لو اختار يقفل: `status='completed'`, `end_reason='manual_stop'`, الوقت اللي اتسجل بيتحسب من ضمن العادة
- لو اختار يكمّل: timer يستأنف من المتبقي
