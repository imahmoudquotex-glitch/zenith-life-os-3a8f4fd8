## B. Phase-Specific Deep Additions — Wave 09 (Tasks + Notes)

### B.1 System DB Strict Usage (No Parallel Tables)

- Tasks/Notes تعيش حصراً في system DBs من W07 (`databases` table مع `is_system=true`).
- ESLint rule + CI scan: ممنوع `CREATE TABLE` جديد بأسماء `tasks|notes|todos` في migrations 09xx.
- FocusLane extension استثناء موثق بـ `TEXT ULID + workspace_id + db_rows` reference.

### B.2 Natural Language Quick Capture (Safe Parser)

- Deterministic parser (لا eval, لا runtime regex من user input).
- Whitelist patterns: `\d+[hd]` للـ duration, `tomorrow|today|next \w+` للـ dates, `#tag` للـ tags, `!high|med|low` للـ priority.
- Arabic equivalents: غدا، اليوم، الإثنين القادم، #علامة، !هام.
- Timezone-aware date resolution via `user.timezone`.
- Max input length 500 chars.
- Fallback: لو parse فشل → side drawer بـ 3 حقول.

### B.3 Recurrence Engine Safety

- RRULE-lite (closed grammar) — لا full iCal RFC.
- Cap: max 100 occurrences per call, max 365 days lookahead.
- DST + timezone + leap year tests إجبارية.
- Idempotent worker: `next_at + last_generated_at` lock via `FOR UPDATE`.
- Audit لكل occurrence generation.

### B.4 Notes Vault & Privacy

- `is_vault BOOLEAN` على note row → blocks AI summary, public share, search index, dashboard widget.
- Server-side enforcement (لا client filter).
- Vault notes searchable عبر client-side index فقط (encrypted على client).
- AI summary على non-vault notes فقط، confirmation modal قبل send.

### B.5 Public Share Hardening

- Token 32-byte URL-safe base64 random.
- TTL default 7d, max 90d.
- Rate-limit creation: 10/h/user, 50/day/user.
- View counter + audit per view.
- Optional password (Argon2id hashed) + optional indexable flag.
- Auto-revoke لو note becomes vault أو archived.

### B.6 Plan Mode Token (Bulk Destructive)

- Token TTL 5min, scope-bound (e.g. `tasks.bulk_delete`).
- Max 100 rows per bulk operation.
- Confirmation dialog يعرض count + sample rows.
- One-time use enforced.
- Audit per operation.

```tsx
export function generateOccurrences(rule: RecurrenceRule, from: Date, to: Date) {
```

### 5.4 Migration 0907 — note_versions prune trigger

```sql
CREATE OR REPLACE FUNCTION prune_note_versions() RETURNS TRIGGER LANGUAGE plpgsql AS $$
BEGIN
  DELETE FROM note_versions
  WHERE note_id = NEW.note_id
    AND id NOT IN (
      SELECT id FROM note_versions
      WHERE note_id = NEW.note_id
      ORDER BY version_no DESC
      LIMIT 50
    );
  RETURN NEW;
END $$;
CREATE TRIGGER trg_note_versions_prune AFTER INSERT ON note_versions
FOR EACH ROW EXECUTE FUNCTION prune_note_versions();
```

### 5.5 Migration 0901 — tasks completed_at trigger

```sql
CREATE OR REPLACE FUNCTION tasks_set_completed_at() RETURNS TRIGGER LANGUAGE plpgsql AS $$
BEGIN
  IF NEW.database_id = (SELECT id FROM databases WHERE workspace_id = NEW.workspace_id AND title='Tasks' AND is_system) THEN
    IF NEW.properties->>'status' = 'done' AND (OLD.properties->>'status') IS DISTINCT FROM 'done' THEN
      NEW.properties = jsonb_set(NEW.properties, '{completed_at}', to_jsonb(now()));
    ELSIF NEW.properties->>'status' IS DISTINCT FROM 'done' THEN
      NEW.properties = NEW.properties - 'completed_at';
    END IF;
  END IF;
  RETURN NEW;
END $$;
CREATE TRIGGER trg_tasks_completed_at BEFORE UPDATE ON db_rows
FOR EACH ROW EXECUTE FUNCTION tasks_set_completed_at();
```

### Migration: 0916_focus_rpcs.sql

```sql
BEGIN;
CREATE OR REPLACE FUNCTION public.start_focus_session(p_user UUID, p_task_id UUID, p_planned_minutes INT DEFAULT 25)
RETURNS UUID LANGUAGE plpgsql SECURITY DEFINER SET search_path = public, pg_temp AS $$
DECLARE v_id UUID;
BEGIN
  IF p_user <> (SELECT auth.uid()) THEN RAISE EXCEPTION 'unauthorized'; END IF;
  UPDATE public.focus_sessions SET status='abandoned', ended_at=now()
  WHERE user_id=p_user AND status='in_progress';
  INSERT INTO public.focus_sessions(user_id, task_id, day_local, started_at, planned_minutes)
  VALUES (p_user, p_task_id, public.local_day(p_user), now(), p_planned_minutes)
  RETURNING id INTO v_id;
  RETURN v_id;
END $$;

CREATE OR REPLACE FUNCTION public.end_focus_session(p_user UUID, p_session_id UUID, p_completed BOOLEAN DEFAULT TRUE)
RETURNS TABLE(duration_minutes INT, xp_awarded INT)
LANGUAGE plpgsql SECURITY DEFINER SET search_path = public, pg_temp AS $$
DECLARE v_started TIMESTAMPTZ; v_min INT; v_xp INT := 0;
BEGIN
  IF p_user <> (SELECT auth.uid()) THEN RAISE EXCEPTION 'unauthorized'; END IF;
  SELECT started_at INTO v_started FROM public.focus_sessions
  WHERE id=p_session_id AND user_id=p_user AND status='in_progress' FOR UPDATE;
  IF NOT FOUND THEN RAISE EXCEPTION 'session_not_found'; END IF;
  v_min := EXTRACT(EPOCH FROM (now() - v_started))::INT / 60;
  UPDATE public.focus_sessions SET ended_at=now(), duration_minutes=v_min,
    status = CASE WHEN p_completed THEN 'completed' ELSE 'abandoned' END
  WHERE id=p_session_id;
  IF p_completed AND v_min >= 15 THEN
    v_xp := LEAST(50, v_min / 5 * 5);
    PERFORM public.award_xp(p_user, 'focus_session', p_session_id, v_xp, 'fs_' || p_session_id::TEXT);
  END IF;
  RETURN QUERY SELECT v_min, v_xp;
END $$;
COMMIT;
```
