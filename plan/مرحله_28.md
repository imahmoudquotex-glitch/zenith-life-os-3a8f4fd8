# المرحلة 28 — محرك الأتمتة الشخصية وسير العمل (Automation & Workflows)

## 🧠 توسعة تنظيمية وتنفيذية شاملة — Automation Operating System

هذه المرحلة لازم تكون واضحة للـ AI/Claude Code كأنها **نظام تشغيل للأتمتة** وليس مجرد صفحة فيها قواعد if-this-then-that. المطلوب هنا بناء engine آمن، قابل للتوسع، قابل للمراقبة، وبدون paywall، يسمح للمستخدم ببناء workflows معقدة بصريًا مع dry-run وtrace وrollback وحماية من loops والـ SSRF وتسريب الـ Vault.

<aside>
🤖

تعليمات تنفيذ للـ AI: ابدأ من الـ domain model والـ runtime contracts قبل الواجهة. لا تكتب UI Builder قبل وجود Rule Schema + Run Engine + Action Registry + Dry-run Trace + Safety Guards.

</aside>

### الهدف النهائي للمرحلة

- تحويل كل أحداث النظام إلى triggers يمكن استخدامها في automations.
- بناء visual workflow builder يفهم trigger → conditions → actions → error paths.
- تشغيل automations عبر worker queue موثوقة وليس مباشرة داخل request.
- توفير dry-run قبل التفعيل يشرح للمستخدم “ما الذي كان سيحدث؟”.
- توفير run history كامل: كل خطوة، input/output، error، duration، retry.
- منع infinite loops وautomation storms.
- منع تشغيل أي user code أو eval أو SQL مباشر.
- جعل `invoke_ai` محكومًا بـ `runAIWithQuota` وDLP وconfirmation للأفعال الخطيرة.
- جعل كل شيء مجاني لكل المستخدمين؛ الحدود فقط لحماية الأداء.

## 🧭 خريطة النظام للـ AI

```
App Domain Event / Scheduled Tick / Manual Button / Incoming Webhook
  ↓
Trigger Resolver
  ↓
Rule Matcher
  ↓
Schema Validator
  ↓
Condition Evaluator
  ↓
Dry-run / Runtime Plan
  ↓
Run Queue
  ↓
Automation Executor Worker
  ↓
Action Registry
  ↓
Audit + Logs + Metrics + Dead Letter
```

### قواعد صارمة

- ممنوع `eval`, `new Function`, arbitrary SQL, أو dynamic import من config.
- ممنوع action يستدعي provider خارجي بدون SSRF guard وtimeout.
- ممنوع AI action يقرأ Vault أو finance raw data.
- ممنوع automation يفعّل automation تاني بلا `root_run_id` وdepth limit.
- ممنوع retry غير محدود.
- ممنوع تغيير صلاحيات المستخدم من automation.
- ممنوع ربط أي automation بميزة مدفوعة أو plan.

## 🧩 Action Registry

```tsx
export type AutomationActionHandler<TConfig> = {
  type: string
  requiredPermission: AutomationPermission
  risky: boolean
  schema: z.ZodType<TConfig>
  dryRun: (ctx: AutomationContext, config: TConfig) => Promise<DryRunStep>
  execute: (ctx: AutomationContext, config: TConfig) => Promise<unknown>
}

export const ACTION_REGISTRY = new Map<string, AutomationActionHandler<any>>()

export function registerAction<T>(handler: AutomationActionHandler<T>) {
  if (ACTION_REGISTRY.has(handler.type)) throw new Error(`DUPLICATE_ACTION:${handler.type}`)
  ACTION_REGISTRY.set(handler.type, handler)
}

export function getAction(type: string) {
  const handler = ACTION_REGISTRY.get(type)
  if (!handler) throw new Error(`UNKNOWN_ACTION:${type}`)
  return handler
}
```

```tsx
export async function executeInvokeAi(ctx: AutomationContext, config: {
  prompt: string
  saveResultTo: string
  requiresConfirmation: boolean
}) {
  assertPermission(ctx, 'ai.run')
  assertAutomationDepth(ctx)
  assertNoVaultOrSecretText(config.prompt)

  const result = await runAIWithQuota({
    workspaceId: ctx.workspaceId,
    userId: ctx.actorUserId,
    purpose: 'automation.invoke_ai',
    prompt: config.prompt,
    dlp: {
      blockVault: true,
      blockSecrets: true,
      redactFinancialSensitive: true,
    },
  })

  if (config.requiresConfirmation) {
    return pendingConfirmationRepo.create({
      workspaceId: ctx.workspaceId,
      runId: ctx.runId,
      variable: config.saveResultTo,
      proposal: result.text,
    })
  }

  ctx.vars[config.saveResultTo] = result.text
  return { savedTo: config.saveResultTo, length: result.text.length }
}
```

## 🔥 Loop Guard + Storm Protection

```tsx
export async function assertRuntimeSafety(ctx: AutomationContext) {
  if (ctx.depth > 10) throw new Error('AUTOMATION_DEPTH_EXCEEDED')
  if (Date.now() - ctx.rootRunStartedAtMs > 60_000) throw new Error('AUTOMATION_ROOT_TIMEOUT')

  const dailyRuns = await automationRepo.countRunsToday(ctx.workspaceId, ctx.actorUserId)
  if (dailyRuns > 1000) throw new Error('AUTOMATION_DAILY_LIMIT_EXCEEDED')

  const workspaceBurst = await redis.incr(`automation:burst:${ctx.workspaceId}`)
  if (workspaceBurst === 1) await redis.expire(`automation:burst:${ctx.workspaceId}`, 60)
  if (workspaceBurst > 5000) throw new Error('AUTOMATION_WORKSPACE_STORM_BLOCKED')
}
```

## 🌐 SSRF Guard للـ Webhooks

```tsx
export async function assertSafeWebhookUrl(rawUrl: string) {
  const url = new URL(rawUrl)
  if (url.protocol !== 'https:') throw new Error('WEBHOOK_HTTPS_REQUIRED')
  const hostname = url.hostname.toLowerCase()
  if (hostname === 'localhost' || hostname.endsWith('.local')) throw new Error('WEBHOOK_LOCAL_BLOCKED')

  const resolved = await dns.promises.lookup(hostname, { all: true })
  for (const record of resolved) {
    if (isPrivateIp(record.address) || isLinkLocalIp(record.address)) {
      throw new Error('WEBHOOK_PRIVATE_IP_BLOCKED')
    }
  }
}
```

## 🧪 Dry-run Trace مفصل

```tsx
export type DryRunTrace = Array<{
  nodeId: string
  step: 'trigger' | 'condition' | 'action' | 'permission' | 'guard'
  status: 'pass' | 'skip' | 'would_execute' | 'blocked' | 'error'
  input?: unknown
  output?: unknown
  reason?: string
}>

export async function dryRunRule(rule: AutomationRule, sampleEvent: Record<string, unknown>) {
  const trace: DryRunTrace = []
  const ctx = createDryRunContext(rule, sampleEvent)

  trace.push({ nodeId: 'trigger', step: 'trigger', status: 'pass', input: sampleEvent })

  for (const permission of rule.permissions) {
    const allowed = await permissionEngine.can(ctx.actorUserId, permission)
    trace.push({ nodeId: permission, step: 'permission', status: allowed ? 'pass' : 'blocked' })
    if (!allowed) return { wouldRun: false, trace }
  }

  for (const cond of rule.conditions) {
    const pass = await conditionEvaluator.evaluate(cond, ctx)
    trace.push({ nodeId: `condition:${cond.left}`, step: 'condition', status: pass ? 'pass' : 'skip', input: cond })
    if (!pass) return { wouldRun: false, trace, reason: 'CONDITION_NOT_MET' }
  }

  for (const action of rule.actions) {
    const handler = getAction(action.type)
    const parsed = handler.schema.parse(action.config)
    const preview = await handler.dryRun(ctx, parsed)
    trace.push({ nodeId: action.type, step: 'action', status: 'would_execute', input: parsed, output: preview })
  }

  return { wouldRun: true, trace }
}
```

## 🏭 Executor Worker منظم

```tsx
export async function executeAutomationRun(runId: string, workerId: string) {
  const run = await automationRepo.leaseRun(runId, workerId)
  if (!run) return { ok: false, reason: 'not_found_or_already_leased' as const }

  const started = Date.now()
  const ctx = await createRuntimeContext(run)

  try {
    await assertRuntimeSafety(ctx)
    const rule = await automationRepo.getRule(run.ruleId)
    const parsed = automationRuleSchema.parse(rule)

    for (const condition of parsed.conditions) {
      const pass = await conditionEvaluator.evaluate(condition, ctx)
      await stepLog.writeCondition(ctx, condition, pass)
      if (!pass) return automationRepo.markSuccess(run.id, { reason: 'condition_not_met' })
    }

    let index = 0
    for (const action of parsed.actions) {
      const handler = getAction(action.type)
      assertPermission(ctx, handler.requiredPermission)

      const stepStart = Date.now()
      try {
        await stepLog.markRunning(ctx, index, action.type)
        const output = await withTimeout(
          handler.execute(ctx, handler.schema.parse(action.config)),
          60_000,
          'ACTION_TIMEOUT',
        )
        await stepLog.markSuccess(ctx, index, action.type, output, Date.now() - stepStart)
      } catch (err) {
        await stepLog.markFailed(ctx, index, action.type, err, Date.now() - stepStart)
        if (action.continueOnError !== true) throw err
      }
      index++
    }

    await automationRepo.markSuccess(run.id, { durationMs: Date.now() - started })
    await automationRepo.incrementStats(run.ruleId, true)
    return { ok: true }
  } catch (err) {
    await automationRepo.markFailed(run.id, normalizeAutomationError(err), Date.now() - started)
    await deadLetterRepo.createIfReplayable(run, err)
    await automationRepo.incrementStats(run.ruleId, false)
    throw err
  }
}
```

## 🎯 نظرة عامة

المرحلة دي بتدّي المستخدم قدرة هائلة: إنه يبني automations خاصة بيه بدون كود. أي حدث في Zenith (تاسك خلصت، عادة اتعملها، حاجة في الكاندر، مبلغ مصروف، وقت معين) يقدر يولّد سلسلة actions تلقائية (إنشاء تاسك جديد، إرسال إشعار، استدعاء AI، تحديث property، إرسال webhook، إنشاء صفحة من template). كل الميزات في Zenith **مجانية بالكامل** بدون أي خطط مدفوعة، والحدود الموجودة بس عشان حماية الأداء وليست لتقييد المستخدمين.

**المستخدم هيكسب إيه:**

- يبني قواعد "لو حصل كذا، اعمل كذا" بصرياً بدون كود.
- يجدول workflows متكررة (يومي، أسبوعي، شهري، cron expression متقدم).
- يستخدم AI داخل الـ automation عشان يلخّص، يصنّف، يقترح، أو يولد محتوى.
- يستورد recipes جاهزة من مكتبة مجتمعية (community automations).
- يشوف سجل تنفيذ كامل لكل automation مع inputs/outputs/errors.
- يوقف، يعمل pause، أو يحذف automation في أي وقت.
- يختبر automation قبل ما يفعّلها (dry run).
- يحمي نفسه من infinite loops بـ guards تلقائية.
- يستخدم variables و expressions داخل الـ actions (task.title, [user.name](http://user.name), now).

**اللي هيتبني في المرحلة دي:**

- جدول `automation_rules` بـ trigger config + actions array + enabled flag + run statistics.
- جدول `automation_runs` لتسجيل كل تنفيذ مع status (pending/running/success/failed) + duration + error.
- جدول `automation_logs` لكل خطوة (action) جوه الـ run بـ input/output JSON.
- جدول `automation_recipes` لمكتبة الـ templates المجتمعية.
- Trigger types: event-based (task.completed, habit.checkin, note.created, expense.added)، scheduled (cron)، conditional (kpi crossed threshold)، manual (user-triggered)، webhook-incoming.
- Action types: create_task، create_note، update_property، send_notification، invoke_ai، send_webhook، create_from_template، add_tag، archive_entity.
- Expression evaluator آمن (sandboxed, no eval) بـ jexl أو expr-eval لتقييم conditions (task.priority == 'high').
- BullMQ executor worker بـ retry + timeout + circuit breaker.
- Infinite loop guard: max 10 nested automations لكل root run.
- Rate limit: 1000 automation runs/يوم لكل مستخدم (حماية أداء فقط، مش plan-based).
- UI flow builder بـ React Flow (nodes + edges + drag/drop).
- Recipe gallery مع تصنيفات (productivity, finance, health, learning, work).
- Dry-run mode بيشغل الـ automation في sandbox بدون ما يطبق التغييرات.

### 5) Expression evaluator (sandboxed)

```tsx
// src/features/automation/lib/expression.ts
import jexl from 'jexl'

//  ...  placeholders
const PLACEHOLDER = /\{\{\s*([^}]+)\s*\}\}/g

export class ExpressionEvaluator {
  private engine: jexl.Jexl

  constructor() {
    this.engine = new jexl.Jexl()
    // Helper functions
    this.engine.addTransform('upper', (s: string) => s?.toUpperCase())
    this.engine.addTransform('lower', (s: string) => s?.toLowerCase())
    this.engine.addTransform('date',  (s: string) => new Date(s).toLocaleDateString('ar-EG'))
    this.engine.addTransform('time',  (s: string) => new Date(s).toLocaleTimeString('ar-EG'))
    this.engine.addTransform('default', (v: unknown, d: unknown) => v ?? d)
  }

  async resolve(template: string, context: Record<string, unknown>): Promise<string> {
    if (!template.includes('{{')) return template
    const matches = Array.from(template.matchAll(PLACEHOLDER))
    let result = template
    for (const m of matches) {
      try {
        const val = await this.engine.eval(m[1], context)
        result = result.replace(m[0], val == null ? '' : String(val))
      } catch {
        // expression failed, leave placeholder
      }
    }
    return result
  }

  async evaluateCondition(
    cond: { left: string; op: string; right: unknown },
    context: Record<string, unknown>,
  ): Promise<boolean> {
    const leftVal = await this.engine.eval(cond.left.replace(/^\{\{|\}\}$/g, ''), context)
    switch (cond.op) {
      case 'eq':      return leftVal === cond.right
      case 'neq':     return leftVal !== cond.right
      case 'gt':      return Number(leftVal) >  Number(cond.right)
      case 'gte':     return Number(leftVal) >= Number(cond.right)
      case 'lt':      return Number(leftVal) <  Number(cond.right)
      case 'lte':     return Number(leftVal) <= Number(cond.right)
      case 'contains':return String(leftVal).includes(String(cond.right))
      case 'in':      return Array.isArray(cond.right) && cond.right.includes(leftVal)
      default:        return false
    }
  }
}
```

### 6) Executor worker (BullMQ)

```tsx
// src/workers/automation-executor.ts
import 'server-only'
import { Worker } from 'bullmq'
import { redisConn } from '@/lib/redis'
import { serviceSupabase } from '@/lib/supabase/service'
import { ExpressionEvaluator } from '@/features/automation/lib/expression'
import { ACTION_CATALOG } from '@/features/automation/actions/catalog'
import { runActionHandler } from '@/features/automation/actions/handlers'
import { log } from '@/lib/monitoring/logger'

const MAX_DEPTH = 10
const RUN_TIMEOUT_MS = 60_000
const DAILY_RUNS_LIMIT = 1000

export const automationWorker = new Worker(
  'automation-runs',
  async (job) => {
    const { runId } = job.data
    const { data: run } = await serviceSupabase.from('automation_runs')
      .select('*, automation_rules(*)').eq('id', runId).single()
    if (!run) return

    // 1. Loop guard
    if (run.depth >= MAX_DEPTH) {
      await failRun(runId, 'MAX_DEPTH_EXCEEDED'); return
    }

    // 2. Daily limit guard
    const today = new Date().toISOString().slice(0, 10)
    const { count } = await serviceSupabase.from('automation_runs')
      .select('*', { count: 'exact', head: true })
      .eq('user_id', run.user_id)
      .gte('created_at', `${today}T00:00:00Z`)
    if ((count ?? 0) > DAILY_RUNS_LIMIT) {
      await failRun(runId, 'DAILY_LIMIT_EXCEEDED'); return
    }

    // 3. Start
    const started = Date.now()
    await serviceSupabase.from('automation_runs')
      .update({ status: 'running', started_at: new Date().toISOString() })
      .eq('id', runId)

    const rule = run.automation_rules
    const evaluator = new ExpressionEvaluator()
    const context: Record<string, unknown> = {
      trigger: run.trigger_event,
      now: new Date().toISOString(),
      user: { id: run.user_id },
      vars: {},
    }

    // 4. Evaluate conditions
    for (const cond of rule.conditions ?? []) {
      const pass = await evaluator.evaluateCondition(cond, context)
      if (!pass) {
        await serviceSupabase.from('automation_runs').update({
          status: 'success',
          completed_at: new Date().toISOString(),
          duration_ms: Date.now() - started,
          error_message: 'CONDITION_NOT_MET',
        }).eq('id', runId)
        return
      }
    }

    // 5. Run actions sequentially
    let stepIdx = 0
    for (const action of rule.actions ?? []) {
      const stepStart = Date.now()
      try {
        const config = ACTION_CATALOG[action.type as keyof typeof ACTION_CATALOG]
        if (!config) throw new Error(`UNKNOWN_ACTION: ${action.type}`)
        // Resolve  expressions  في الـ config
        const resolved = await resolveDeep(action.config, context, evaluator)
        const parsed = config.config.parse(resolved)
        const output = await Promise.race([
          runActionHandler(action.type, parsed, run.user_id, context),
          timeoutPromise(RUN_TIMEOUT_MS),
        ])
        await serviceSupabase.from('automation_logs').insert({
          run_id: runId, user_id: run.user_id, step_index: stepIdx,
          action_type: action.type, action_config: parsed,
          input: context, output, status: 'success',
          duration_ms: Date.now() - stepStart,
        })
        // store output في context
        if (action.saveTo) context.vars[action.saveTo] = output
      } catch (err) {
        await serviceSupabase.from('automation_logs').insert({
          run_id: runId, user_id: run.user_id, step_index: stepIdx,
          action_type: action.type, action_config: action.config,
          status: 'failed', error_message: String(err),
          duration_ms: Date.now() - stepStart,
        })
        if (action.continueOnError !== true) {
          await failRun(runId, String(err), Date.now() - started); return
        }
      }
      stepIdx++
    }

    // 6. Success
    await serviceSupabase.from('automation_runs').update({
      status: 'success',
      completed_at: new Date().toISOString(),
      duration_ms: Date.now() - started,
    }).eq('id', runId)
    await serviceSupabase.rpc('incr_automation_stats', {
      p_rule_id: rule.id, p_success: true,
    })
  },
  { connection: redisConn, concurrency: 10 }
)

async function failRun(runId: string, err: string, duration?: number) {
  await serviceSupabase.from('automation_runs').update({
    status: 'failed', error_message: err,
    completed_at: new Date().toISOString(),
    duration_ms: duration ?? null,
  }).eq('id', runId)
}

function timeoutPromise(ms: number) {
  return new Promise((_, rej) => setTimeout(() => rej(new Error('TIMEOUT')), ms))
}

async function resolveDeep(
  obj: unknown, ctx: Record<string, unknown>, ev: ExpressionEvaluator,
): Promise<unknown> {
  if (typeof obj === 'string') return ev.resolve(obj, ctx)
  if (Array.isArray(obj)) return Promise.all(obj.map(x => resolveDeep(x, ctx, ev)))
  if (obj && typeof obj === 'object') {
    const out: Record<string, unknown> = {}
    for (const [k, v] of Object.entries(obj)) out[k] = await resolveDeep(v, ctx, ev)
    return out
  }
  return obj
}
```

### 7) Action handlers

```tsx
// src/features/automation/actions/handlers.ts
import 'server-only'
import { serviceSupabase } from '@/lib/supabase/service'
import { sendNotification } from '@/lib/notifications/send'
import { invokeAI } from '@/lib/ai/invoke'
import { dispatchWebhook } from '@/lib/webhooks/dispatch'

export async function runActionHandler(
  type: string, config: any, userId: string, context: Record<string, unknown>,
): Promise<unknown> {
  switch (type) {
    case 'create_task':
      return serviceSupabase.from('tasks').insert({
        user_id: userId, title: config.title,
        priority: config.priority, due_date: config.dueDate,
        project_id: config.projectId,
      }).select().single()

    case 'create_note':
      // Vault guard: لو is_vault=true، نسجل audit
      return serviceSupabase.from('notes').insert({
        user_id: userId, title: config.title,
        content: config.content, is_vault: config.isVault,
      }).select().single()

    case 'update_property':
      return serviceSupabase.from(`${config.entityType}s`)
        .update({ [config.property]: config.value })
        .eq('id', config.entityId).eq('user_id', userId)

    case 'send_notification':
      return sendNotification(userId, {
        title: config.title, body: config.body,
        channel: config.channel, link: config.url,
      })

    case 'invoke_ai':
      // Vault context guard
      const result = await invokeAI({
        userId, prompt: config.prompt, model: config.model,
      })
      return result.text

    case 'send_webhook':
      return dispatchWebhook(userId, 'automation.outgoing', {
        url: config.url, method: config.method,
        headers: config.headers, body: config.body,
      })

    case 'create_from_template':
      return serviceSupabase.rpc('create_page_from_template', {
        p_user: userId, p_template: config.templateId,
        p_overrides: config.overrides ?? {},
      })

    case 'add_tag':
      // upsert tags
      for (const tag of config.tags) {
        await serviceSupabase.from('entity_tags').upsert({
          user_id: userId, entity_type: config.entityType,
          entity_id: config.entityId, tag,
        })
      }
      return { added: config.tags.length }

    case 'archive_entity':
      return serviceSupabase.from(`${config.entityType}s`)
        .update({ is_archived: true })
        .eq('id', config.entityId).eq('user_id', userId)

    case 'wait':
      await new Promise(r => setTimeout(r, config.durationMs))
      return { waited: config.durationMs }

    default:
      throw new Error(`UNKNOWN_ACTION: ${type}`)
  }
}
```

### 10) Dry-run mode (محاكاة بدون تطبيق)

```tsx
// src/features/automation/lib/dry-run.ts
import { ExpressionEvaluator } from './expression'

export async function dryRunAutomation(
  rule: { trigger_config: unknown; conditions: any[]; actions: any[] },
  sampleEvent: Record<string, unknown>,
) {
  const evaluator = new ExpressionEvaluator()
  const context = { trigger: sampleEvent, now: new Date().toISOString(), vars: {} }
  const trace: Array<{ step: string; detail: unknown }> = []

  // Conditions
  for (const cond of rule.conditions ?? []) {
    const pass = await evaluator.evaluateCondition(cond, context)
    trace.push({ step: 'condition', detail: { cond, pass } })
    if (!pass) return { wouldRun: false, trace, reason: 'CONDITION_NOT_MET' }
  }

  // Actions (resolve only, don't execute)
  for (const action of rule.actions ?? []) {
    const resolved = await resolveAllStrings(action.config, context, evaluator)
    trace.push({ step: 'action', detail: { type: action.type, resolved } })
  }

  return { wouldRun: true, trace }
}

async function resolveAllStrings(obj: any, ctx: any, ev: ExpressionEvaluator): Promise<any> {
  if (typeof obj === 'string') return ev.resolve(obj, ctx)
  if (Array.isArray(obj)) return Promise.all(obj.map(x => resolveAllStrings(x, ctx, ev)))
  if (obj && typeof obj === 'object') {
    const out: any = {}
    for (const [k, v] of Object.entries(obj)) out[k] = await resolveAllStrings(v, ctx, ev)
    return out
  }
  return obj
}
```

```tsx
import { useState, useCallback } from 'react'
import ReactFlow, {
  Background, Controls, MiniMap,
  addEdge, applyNodeChanges, applyEdgeChanges,
  type Node, type Edge,
} from 'reactflow'
import { TriggerNode } from './nodes/TriggerNode'
import { ActionNode }  from './nodes/ActionNode'
import { ConditionNode } from './nodes/ConditionNode'

const nodeTypes = { trigger: TriggerNode, action: ActionNode, condition: ConditionNode }

export function FlowBuilder({ rule, onChange }: { rule: any; onChange: (r: any) => void }) {
  const [nodes, setNodes] = useState<Node[]>(rule.nodes ?? [])
  const [edges, setEdges] = useState<Edge[]>(rule.edges ?? [])

  return (
    <div className="h-[600px] border rounded-lg">
      <ReactFlow
        nodes={nodes} edges={edges}
        nodeTypes={nodeTypes}
        onNodesChange={(ch) => setNodes(n => applyNodeChanges(ch, n))}
        onEdgesChange={(ch) => setEdges(e => applyEdgeChanges(ch, e))}
        onConnect={(c) => setEdges(e => addEdge(c, e))}
        fitView
      >
        <Background />
        <Controls />
        <MiniMap />
      </ReactFlow>
    </div>
  )
}
```
