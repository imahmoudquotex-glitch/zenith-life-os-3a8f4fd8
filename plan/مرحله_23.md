## 🛠️ Technical Implementation

### Migration 0022 — `voice_memos`

```sql
CREATE TABLE voice_memos (
  id TEXT PRIMARY KEY DEFAULT 'aud_' || lower(hex(random_bytes(10))),
  page_id TEXT REFERENCES pages(id) ON DELETE CASCADE,
  user_id UUID REFERENCES profiles(id),
  
  -- Storage Info
  storage_path TEXT NOT NULL, -- S3/R2 path
  duration_seconds INT NOT NULL,
  file_size_bytes BIGINT NOT NULL,
  
  -- AI Transcription
  transcript TEXT,
  summary TEXT,
  action_items JSONB, -- Array of strings
  transcription_status TEXT CHECK (transcription_status IN ('pending', 'processing', 'completed', 'failed')),
  
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE INDEX idx_voice_memos_page ON voice_memos(page_id);
```

### The "Likely Favorite" Workflow

1.  **Instant Record:** زرار أحمر عائم في الموبايل أو `CMD+K -> Record`.
2.  **Background Upload:** التسجيل يترفع لـ Cloudflare R2 (أرخص وأسرع).
3.  **Whisper Hook:** Trigger لـ Edge Function تبعت الملف لـ OpenAI Whisper API.
4.  **AI Post-Process:** بعد الـ transcript، يتبعت الـ text لـ Mistral 7B يعمل "Refined Note":
    - يحول الكلام لـ bullet points.
    - يستخرج الـ dates والمواعيد.
    - يعمل auto-tagging.

### UI Specs (Zenith Style)

- **Waveform Visualization:** أثناء التسجيل وبعده.
- **Transcript Highlighter:** الكلام يتظلل وهو بيتقري (Karaoke style).
- **One-Click Convert:** زرار "Convert to Page" يحول الـ summary لـ full page blocks.
