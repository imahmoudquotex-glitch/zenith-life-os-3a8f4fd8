## 🧠 Master Architect Upgrade — Geo Residency Gate

- **Residency-as-attribute:** كل workspace له `region` (us-east-1, eu-west-1, me-south-1). كل data physical/logical موضعها mandatory.
- **Active-Passive DR:** primary region writes + reads. passive region read-replica + cold storage. RTO ≤ 1h، RPO ≤ 15min.
- **Cross-region forbidden by default:** queries عابرة للمناطق تتطلب explicit `allowCrossRegion: true` + audit log.
- **Edge routing:** Cloudflare Workers يـ route requests للـ home region بناءً على workspace_id (cookie lookup).
- **Vault keys regional:** vault key shards (Shamir 3-of-5) stored within tenant's region — لا تتجاوز الحدود.
- **Compliance mapping:** EU=GDPR strict, US=CCPA, MENA=local PDPL where applicable.
- **Backup encryption:** كل backup بـ AES-256-GCM + region-local KMS key.

### Edge Routing (Cloudflare Worker)

```tsx
// /infra/edge/router.ts
export default {
	async fetch(req: Request, env: Env) {
		const url = new URL(req.url)
		const workspaceId = await resolveWorkspaceFromRequest(req, env)
		const region = await env.KV.get(`ws_region:${workspaceId}`) ?? 'us-east-1'

		const regionalOrigin = {
			'us-east-1':    'https://us.lifeos.app',
			'eu-west-1':    'https://eu.lifeos.app',
			'me-south-1':   'https://me.lifeos.app',
			'ap-southeast-1':'https://ap.lifeos.app',
		}[region]

		const proxied = new Request(`${regionalOrigin}${url.pathname}${url.search}`, req)
		proxied.headers.set('x-workspace-region', region)
		return fetch(proxied)
	},
}
```

### Cross-Region Query Guard

```tsx
// /src/lib/guards/region.ts
import { AsyncLocalStorage } from 'node:async_hooks'
import { logger } from '@/observability/logger'

const regionContext = new AsyncLocalStorage<{ region: string; workspaceId: string }>()

export function withRegion<T>(region: string, workspaceId: string, fn: () => Promise<T>) {
	return regionContext.run({ region, workspaceId }, fn)
}

export async function assertSameRegion(targetEntityRegion: string, opts?: { allowCrossRegion?: boolean; reason?: string }) {
	const ctx = regionContext.getStore()
	if (!ctx) throw new Error('No region context')
	if (ctx.region === targetEntityRegion) return
	if (!opts?.allowCrossRegion) {
		throw new Error(`Cross-region access denied: ${ctx.region} → ${targetEntityRegion}`)
	}
	await db.query(`
		INSERT INTO cross_region_access_log (user_id, workspace_id, source_region, target_region, reason)
		VALUES ($1, $2, $3, $4, $5)
	`, [currentUserId(), ctx.workspaceId, ctx.region, targetEntityRegion, opts.reason ?? 'unspecified'])
	logger.warn({ event: 'cross_region_access', ...ctx, targetEntityRegion })
}
```

### DR Replication Monitor

```tsx
// /workers/dr-monitor.ts (runs every minute)
export async function checkReplicationLag() {
	for (const region of ['us-east-1','eu-west-1','me-south-1']) {
		const lag = await measureReplicationLagSeconds(region)
		await db.query(`
			UPDATE dr_replication_state
			SET lag_seconds = $1, last_verified_at = now()
			WHERE region = $2
		`, [lag, region])
		if (lag > 900) {  // 15 min RPO breach
			await pageOnCall('dr_rpo_breach', { region, lag })
		}
	}
}
```

### Failover Procedure (documented runbook)

```markdown
## DR Failover: Promote Passive → Primary

### Pre-conditions
- [ ] Confirmed primary region down ≥ 5 min
- [ ] Replication lag < 60s at time of incident
- [ ] On-call commander assigned

### Steps (target: 60 min)
1. **T+0:** Activate incident war room
2. **T+5:** Verify passive replica health (last LSN, schema parity)
3. **T+10:** Promote replica to primary (Supabase support ticket / managed runbook)
4. **T+20:** Update Cloudflare KV: `ws_region:*` → new primary
5. **T+30:** Smoke test: read + write + AI call + realtime connect
6. **T+45:** Enable writes, announce status page
7. **T+60:** Begin RCA notes

### Post-failover
- Decommission old primary as new passive (when restored)
- Replay any in-flight ops from outbox
- Validate no data loss vs RPO target
```

- Workspace creation prompt asks for region; immutable after first write (or one-time migration with explicit confirmation).
- Zero cross-region queries in audit log without `allowCrossRegion=true` + audit.
- Replication lag < 60s p99 سنة.
- Failover drill quarterly، RTO ≤ 1h من end-to-end.
- Vault key shards never cross region boundaries (cryptographic verification).
- Status page shows per-region health.
