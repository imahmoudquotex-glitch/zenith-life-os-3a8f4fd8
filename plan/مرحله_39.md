### Course Graph Service

```tsx
export async function loadCourseGraph(courseId: string, ctx: RequestContext) {
  const course = await courseRepo.getOwned(courseId, ctx.userId)
  const modules = await moduleRepo.listByCourse(course.id)
  const lessons = await lessonRepo.listByCourse(course.id)
  return modules.map((m) => ({
    ...m,
    lessons: lessons.filter((l) => l.moduleId === m.id).sort((a, b) => a.position - b.position),
  }))
}
```

### Flashcard Review Mutation

```tsx
export async function reviewFlashcard(cardId: string, quality: number, ctx: RequestContext) {
  const card = await flashcardRepo.getOwned(cardId, ctx.userId)
  const patch = sm2Update(card, quality)
  await flashcardRepo.update(card.id, patch)
  await outbox.publish('learning.flashcard_reviewed', { userId: ctx.userId, cardId, quality })
  return patch
```

### Acceptance

- أي AI output بدون citations لا يتم حفظه.
- استيراد الكورسات يخزن metadata/notes فقط ولا يسرق فيديوهات أو محتوى مدفوع.
- Learning analytics ذاتية فقط، مفيش leaderboards.

## 🧠 Master Architect Upgrade — Learning Fairness & AI Safety Gate

- **Free learning invariant:** لا premium courses، لا paid certificates، ولا AI learning tools خلف paywall. rate limits لحماية الأداء فقط.
- **No comparison:** analytics ذاتية فقط: هذا الأسبوع مقابل السابق، current pace مقابل هدف المستخدم. ممنوع leaderboards أو "طلاب آخرين أسرع منك".
- **AI citations:** AI summaries/flashcards/roadmaps تعرض source lesson/note references، وتسمح للمستخدم بتعديل/رفض قبل الحفظ.
- **Copyright/import safety:** YouTube/Udemy/Coursera import يحترم terms، يخزن metadata/notes لا pirated video/content، ويستخدم user-provided exports حيث يلزم.
- **Vault/privacy:** learning RAG على course notes فقط، no vault, no private finance/journal unless user explicitly includes summarized context.

### SM-2 Algorithm

- `ease_factor` calculation يتنفذ في كل الحالات.
- لو quality أقل من 3، interval يرجع لـ 1، لكن `ease_factor` يتعدل برضو.

### SM-2 Fixed Algorithm (canonical)

```tsx
// /src/lib/learning/sm2.ts
export function sm2Update(card: Flashcard, quality: number /* 0-5 */) {
  let { ease_factor: ef, interval_days: I, repetitions: n } = card

  // ef calculation runs ALWAYS, not just q>=3
  ef = Math.max(1.3, ef + (0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02)))

  if (quality < 3) {
    // Failed recall — reset interval but KEEP updated ef
    n = 0
    I = 1
  } else {
    // Successful recall
    if (n === 0) I = 1
    else if (n === 1) I = 6
    else I = Math.round(I * ef)
    n += 1
  }

  const next_review_at = addDays(new Date(), I)
  return {
    ease_factor: ef,
    interval_days: I,
    repetitions: n,
    next_review_at,
    last_quality: quality,
    last_reviewed_at: new Date(),
  }
}
```

### Mood × Study Correlation (مكمل)

```sql
-- /src/lib/learning/queries/mood-study-correlation.sql
SELECT
  date_trunc('day', s.started_at) AS day,
  AVG(s.actual_focused_seconds / 60.0) AS study_minutes,
  AVG(m.mood_score) AS avg_mood,
  COUNT(*) AS sessions
FROM focus_sessions_v2 s
LEFT JOIN mood_logs m ON date_trunc('day', m.logged_at) = date_trunc('day', s.started_at)
  AND m.user_id = s.user_id
WHERE s.user_id = $1 AND s.linked_course_id IS NOT NULL
  AND s.started_at > now() - INTERVAL '90 days'
GROUP BY day
ORDER BY day;
```

## 🎯 نظرة عامة

قسم متكامل لتسجيل وإدارة كل كورس بيتعلمه المستخدم — سواء كورس مشتراه من Udemy/Coursera، أو فيديو يوتيوب طويل، أو كتاب، أو مسار تعلم ذاتي. مش مجرد "list of courses" — ده محرك تعلم كامل بيحلل سرعتك، يفكرك بالـ spaced repetition، يخرّج لك flashcards من ملاحظاتك بالـ AI، ويربط الكورس بأهدافك في W40 وبجلسات التركيز في W43.

**المستخدم هيكسب إيه:**

- يسجل كل كورس بياخده في مكان واحد مع progress tracking دقيق لكل درس
- AI بيلخص أي درس/فيديو ويطلع منه flashcards و mind map تلقائي
- Spaced Repetition (Anki-style) تلقائي لأي معلومة محفوظها
- بيحسب "كم يوم متبقي" لإنهاء الكورس بناءً على سرعته الحالية
- يربط الكورس بـ Focus Session (W43) — اضغط "ابدأ مذاكرة" يفتح الـ Pomodoro فوراً
- Certificates wall — كل شهادة بيخلصها تتعرض ك trophy
- Notes في Markdown مع AI Q&A على المحتوى
- Study Streak خاص بالتعلم منفصل عن streak اليومي
- يقارن نفسه مع نفسه فقط (مفيش leaderboard مع ناس تانية — anti-comparison policy)
- Roadmap Builder بالـ AI: "عايز أتعلم Next.js" → يطلع له خطة 30 يوم
- استيراد من Udemy/Coursera/YouTube بـ URL واحد

**Feature 5 — Spaced Repetition Engine (SM-2)**

```tsx
// /src/services/learning/sm2.ts
// Quality: 0=fail .. 5=perfect
export function sm2(card: Flashcard, q: number) {
  let { ease_factor: ef, interval_days: I, repetitions: n } = card
  ef = Math.max(1.3, ef + (0.1 - (5 - q) * (0.08 + (5 - q) * 0.02)))
  if (q < 3) { n = 0; I = 1 }
  else {
    n += 1
    if (n === 1) I = 1
    else if (n === 2) I = 6
    else I = Math.round(I * ef)
  }
  const next = new Date(); next.setDate(next.getDate() + I)
  return { ease_factor: ef, interval_days: I, repetitions: n, next_review_at: next, last_quality: q }
}
```

**Feature 6 — Pace Predictor**

```tsx
// Daily cron at 03:00
export async function predictAllPaces() {
  const active = await db.from('courses').select('*').eq('status', 'active')
  for (const c of active.data ?? []) {
    const last14 = await lessonsCompletedLast14Days(c.id)
    const dailyRate = last14 / 14
    const remaining = c.total_lessons - c.completed_lessons
    const daysToFinish = dailyRate > 0 ? Math.ceil(remaining / dailyRate) : null
    const predicted = daysToFinish ? addDays(new Date(), daysToFinish) : null
    await db.from('courses').update({ predicted_finish_date: predicted }).eq('id', c.id)

    // If predicted > target by >30%, send AI nudge
    if (predicted && c.target_finish_date && daysOver(predicted, c.target_finish_date) > 0.3) {
      await sendGentleNudge(c.user_id, `لو كملت بنفس السرعة، هتخلص الكورس بعد ${daysToFinish} يوم — متأخر شوية عن هدفك`)
    }
  }
}
```
