# W23.6 — Realtime CRDT Engine (Yjs + Hocuspocus dedicated server)

## 🧠 توسعة تنفيذية كاملة — Realtime Collaboration Operating System

مرحلة 30 لازم تتحول من “تشغيل Yjs + Hocuspocus” إلى **نظام تشغيل كامل للتعاون اللحظي**: WebSocket dedicated server، CRDT persistence، awareness، offline queue، conflict-free undo/redo، E2EE compatibility، load testing، disaster recovery، observability، وحماية صارمة ضد cross-tenant access.

<aside>
⚡

تعليمات للـ AI/Claude Code: لا تستخدم Supabase Realtime لتعديل المستندات التعاونية الثقيلة. Supabase يبقى للـ metadata والـ audit، أما محتوى التحرير اللحظي فمساره الوحيد هو Hocuspocus/Yjs dedicated server.

</aside>

### الهدف النهائي

- دعم 50+ محرر في نفس الوثيقة بدون lag ملحوظ.
- فصل collaboration traffic عن Next.js/Supabase حتى لا ينهار الـ API تحت WebSocket load.
- تخزين Yjs binary state + ops log لاستعادة أي وثيقة.
- جعل presence/cursors ephemeral وغير مخزنة.
- حماية كل document namespace بـ workspace/user/entity permission.
- دعم وثائق Vault بـ opaque encrypted updates فقط.
- توفير offline editing queue وإعادة مزامنة بعد reconnect.
- تقديم metrics دقيقة: latency، ops/sec، connection count، update size، compaction ratio.

### ممنوعات صارمة

- ممنوع تخزين awareness/cursors في DB.
- ممنوع إرسال content plaintext لو الوثيقة Vault/E2EE.
- ممنوع قبول documentName لا يبدأ بـ `ws_{workspaceId}/`.
- ممنوع الاعتماد على `postgres_changes` لتحرير المستند.
- ممنوع logging للـ update binary payload.
- ممنوع فتح WebSocket بدون JWT صالح.
- ممنوع تحميل document state من workspace آخر حتى لو doc_id صحيح.

## 🔐 Document Name Contract

```tsx
export function buildDocumentName(args: {
  workspaceId: string
  entityType: RealtimeEntityType
  entityId: string
}) {
  return `ws_${args.workspaceId}/${args.entityType}_${args.entityId}`
}

export function parseDocumentName(documentName: string) {
  const match = documentName.match(/^ws_([^/]+)\/(page|whiteboard|database_view|doc)_(.+)$/)
  if (!match) throw new Error('INVALID_DOCUMENT_NAME')
  return {
    workspaceId: match[1],
    entityType: match[2] as RealtimeEntityType,
    entityId: match[3],
  }
}

export function assertDocumentMatchesJwt(documentName: string, jwtWorkspaceId: string) {
  const parsed = parseDocumentName(documentName)
  if (parsed.workspaceId !== jwtWorkspaceId) throw new Error('REALTIME_CROSS_WORKSPACE_BLOCKED')
  return parsed
}
```

## 🔑 Auth + Entity Permission

```tsx
export async function authenticateRealtimeConnection(args: {
  token: string
  documentName: string
}): Promise<RealtimeAuthContext> {
  const payload = await verifySupabaseJwt(args.token)
  const parsed = assertDocumentMatchesJwt(args.documentName, payload.workspace_id)

  const permission = await entityPermissionRepo.resolve({
    workspaceId: parsed.workspaceId,
    userId: payload.sub,
    entityType: parsed.entityType,
    entityId: parsed.entityId,
  })

  if (!permission || !['read', 'comment', 'edit'].includes(permission)) {
    throw new Error('REALTIME_PERMISSION_DENIED')
  }

  return {
    workspaceId: parsed.workspaceId,
    userId: payload.sub,
    sessionId: payload.session_id ?? crypto.randomUUID(),
    permission: permission as RealtimePermission,
    isE2ee: await realtimeRepo.isDocumentE2ee(args.documentName),
  }
}
```

## ⚙️ Hocuspocus Server محسن

```tsx
import { Server } from '@hocuspocus/server'
import * as Y from 'yjs'

export const realtimeServer = Server.configure({
  port: Number(process.env.PORT ?? 3001),
  timeout: 30_000,
  debounce: 750,
  maxDebounce: 10_000,

  async onAuthenticate({ token, documentName }) {
    if (!token) throw new Error('REALTIME_TOKEN_REQUIRED')
    const ctx = await authenticateRealtimeConnection({ token, documentName })
    await connectionRepo.open({
      workspaceId: ctx.workspaceId,
      docName: documentName,
      userId: ctx.userId,
      sessionId: ctx.sessionId,
      permission: ctx.permission,
    })
    return { user: ctx }
  },

  async onLoadDocument({ documentName, context }) {
    const persisted = await yjsStore.load(documentName, context.user.workspaceId)
    const doc = new Y.Doc()
    if (persisted?.state) Y.applyUpdate(doc, persisted.state)
    return doc
  },

  async onChange({ documentName, update, context }) {
    if (context.user.permission !== 'edit') throw new Error('REALTIME_EDIT_PERMISSION_REQUIRED')
    await opsLimiter.assertAllowed(`${context.user.userId}:${documentName}`)
    await opaqueStorePolicy.assertNoPlaintextLeak({ update, isE2ee: context.user.isE2ee })

    await opsLog.append({
      workspaceId: context.user.workspaceId,
      docName: documentName,
      clientId: context.connection.clientId,
      update,
      origin: 'remote',
    })

    metrics.increment('realtime_yjs_update_total', { workspaceId: context.user.workspaceId })
    metrics.observe('realtime_yjs_update_bytes', update.byteLength)
  },

  async onStoreDocument({ documentName, document, context }) {
    const state = Y.encodeStateAsUpdate(document)
    const stateVector = Y.encodeStateVector(document)
    await yjsStore.upsert({
      workspaceId: context.user.workspaceId,
      docName: documentName,
      state,
      stateVector,
      sizeBytes: state.byteLength,
    })
  },

  async onDisconnect({ documentName, context }) {
    await connectionRepo.close({
      workspaceId: context.user.workspaceId,
      docName: documentName,
      userId: context.user.userId,
      sessionId: context.user.sessionId,
      reason: 'disconnect',
    })
  },
})
```

## 🛡️ Backpressure Strategy

```tsx
export class RealtimeBackpressure {
  async classifyLoad() {
    const memory = process.memoryUsage().heapUsed / process.memoryUsage().heapTotal
    const eventLoopDelay = await metrics.currentEventLoopDelayMs()
    if (memory > 0.9 || eventLoopDelay > 250) return 'critical'
    if (memory > 0.75 || eventLoopDelay > 100) return 'high'
    return 'normal'
  }

  async shouldDropAwareness() {
    const load = await this.classifyLoad()
    return load === 'high' || load === 'critical'
  }

  async shouldRejectNewConnection() {
    const load = await this.classifyLoad()
    return load === 'critical'
  }
}
```

### قواعد الضغط

- awareness/cursors أول شيء يتم throttling/drop.
- edits لا تُفقد؛ إما تقبل وتُسجل في ops log أو ترفض برسالة retryable.
- عند critical load يتم رفض connections جديدة مؤقتًا.
- أي document يزيد حجمه عن حد معين يدخل compaction queue.

## 🔒 E2EE Compatibility Layer

```tsx
export async function encodeRealtimeUpdate(args: {
  update: Uint8Array
  isVault: boolean
  encrypt: (data: Uint8Array) => Promise<Uint8Array>
}) {
  if (!args.isVault) return { encrypted: false, data: args.update }
  return { encrypted: true, data: await args.encrypt(args.update) }
}

export async function decodeRealtimeUpdate(args: {
  data: Uint8Array
  encrypted: boolean
  decrypt: (data: Uint8Array) => Promise<Uint8Array>
}) {
  if (!args.encrypted) return args.data
  return args.decrypt(args.data)
}
```

- server لا يفك تشفير updates.

## 🧑‍💻 Client Provider + Offline Queue

```tsx
export function useRealtimeDocument(args: {
  workspaceId: string
  entityType: RealtimeEntityType
  entityId: string
  token: string
  isVault: boolean
}) {
  const [status, setStatus] = useState<RealtimeConnectionStatus>('connecting')
  const docRef = useRef<Y.Doc>()
  const providerRef = useRef<HocuspocusProvider>()

  useEffect(() => {
    const doc = new Y.Doc()
    const provider = new HocuspocusProvider({
      url: env.NEXT_PUBLIC_REALTIME_URL,
      name: buildDocumentName(args),
      document: doc,
      token: args.token,
      connect: true,
      onStatus: ({ status }) => setStatus(status as RealtimeConnectionStatus),
      onAuthenticationFailed: () => setStatus('failed'),
    })

    docRef.current = doc
    providerRef.current = provider

    return () => {
      provider.destroy()
      doc.destroy()
    }
  }, [args.workspaceId, args.entityId, args.token])

  return {
    doc: docRef.current,
    provider: providerRef.current,
    awareness: providerRef.current?.awareness,
    status,
  }
}
```

## 🧰 Persistence Store

```tsx
export const yjsStore = {
  async load(docName: string, workspaceId: string) {
    return db.oneOrNone(`
      SELECT state, state_vector, size_bytes
      FROM yjs_documents
      WHERE doc_name=$1 AND workspace_id=$2
    `, [docName, workspaceId])
  },

  async upsert(args: {
    workspaceId: string
    docName: string
    state: Uint8Array
    stateVector: Uint8Array
    sizeBytes: number
  }) {
    const parsed = parseDocumentName(args.docName)
    await db.none(`
      INSERT INTO yjs_documents(doc_name, workspace_id, entity_type, entity_id, state, state_vector, size_bytes, updated_at)
      VALUES($1,$2,$3,$4,$5,$6,$7,now())
      ON CONFLICT(doc_name) DO UPDATE SET
        state=EXCLUDED.state,
        state_vector=EXCLUDED.state_vector,
        size_bytes=EXCLUDED.size_bytes,
        updated_at=now()
    `, [args.docName, args.workspaceId, parsed.entityType, parsed.entityId, args.state, args.stateVector, args.sizeBytes])
  },
}
```

## 🧯 DR Replay Service

```tsx
export async function restoreDocumentFromOps(args: {
  workspaceId: string
  docName: string
  sinceSnapshotId?: string
}) {
  const snapshot = await snapshotRepo.latest(args.workspaceId, args.docName)
  const doc = new Y.Doc()

  if (snapshot) Y.applyUpdate(doc, snapshot.state)

  const updates = await opsLog.listAfter({
    workspaceId: args.workspaceId,
    docName: args.docName,
    after: snapshot?.createdAt,
  })

  for (const u of updates) {
    verifyChecksum(u.update_data, u.checksum_sha256)
    Y.applyUpdate(doc, u.update_data)
  }

  const restored = Y.encodeStateAsUpdate(doc)
  await yjsStore.upsert({
    workspaceId: args.workspaceId,
    docName: args.docName,
    state: restored,
    stateVector: Y.encodeStateVector(doc),
    sizeBytes: restored.byteLength,
  })

  return { restoredBytes: restored.byteLength, updatesApplied: updates.length }
}
```

## 🧠 Master Architect Upgrade — Realtime Scaling Gate

- **Dedicated WS server:** Hocuspocus (Node) على [Fly.io/Railway](http://Fly.io/Railway) مفصول عن Supabase. WebSocket persistent connection، not HTTP polling.
- **Tenant isolation:** كل document namespace `ws_{workspace_id}/doc_{doc_id}`. server يرفض connection بدون JWT يحوي workspace_id matching.
- **E2EE compatibility:** للـ vault docs، Yjs update payloads encrypted client-side قبل send. server يخزن opaque blobs ولا يقرأ content.
- **Awareness ephemeral:** presence/cursors لا تـ persist، تنتهي عند disconnect.
- **Backpressure:** rate limit updates per client (50 ops/sec). drop awareness messages أولاً تحت ضغط.
- **Persistence:** Yjs document state stored as binary in postgres `yjs_documents` table، snapshot كل 100 updates أو 60s، compacted nightly.
- **DR:** snapshot + ops log قابلين للـ replay لاستعادة كامل لأي وثيقة.

### Hocuspocus Server

```tsx
// /servers/realtime/index.ts
import { Server } from '@hocuspocus/server'
import { Database } from '@hocuspocus/extension-database'
import { Logger } from '@hocuspocus/extension-logger'
import jwt from 'jsonwebtoken'
import * as Y from 'yjs'

const server = Server.configure({
	port: 3001,
	timeout: 30_000,

	async onAuthenticate({ token, documentName, requestParameters }) {
		const payload = jwt.verify(token, process.env.SUPABASE_JWT_SECRET!) as any
		const { workspace_id, sub: user_id } = payload

		// Document name MUST match workspace
		if (!documentName.startsWith(`ws_${workspace_id}/`)) {
			throw new Error('Cross-workspace document access denied')
		}

		// Check entity-level permission
		const entityId = documentName.split('/doc_')[1]
		const canEdit = await checkEntityPermission(user_id, entityId, 'edit')
		if (!canEdit) throw new Error('Permission denied')

		return { user: { id: user_id, workspaceId: workspace_id } }
	},

	async onLoadDocument({ documentName }) {
		const row = await db.query('SELECT state FROM yjs_documents WHERE doc_name = $1', [documentName])
		if (row.rows[0]) {
			const doc = new Y.Doc()
			Y.applyUpdate(doc, row.rows[0].state)
			return doc
		}
		return new Y.Doc()
	},

	async onStoreDocument({ documentName, document, context }) {
		const state = Y.encodeStateAsUpdate(document)
		const stateVector = Y.encodeStateVector(document)
		const [workspaceId, entityPart] = documentName.split('/')
		const workspace = workspaceId.replace('ws_', '')

		await db.query(`
			INSERT INTO yjs_documents (doc_name, workspace_id, entity_type, entity_id, state, state_vector, size_bytes, updated_at)
			VALUES ($1, $2, 'page', $3, $4, $5, $6, now())
			ON CONFLICT (doc_name) DO UPDATE SET
				state = EXCLUDED.state,
				state_vector = EXCLUDED.state_vector,
				size_bytes = EXCLUDED.size_bytes,
				updated_at = now()
		`, [documentName, workspace, entityPart.replace('doc_', ''), state, stateVector, state.byteLength])
	},

	async onChange({ documentName, update, context }) {
		// Append to ops log for DR replay
		await db.query(`
			INSERT INTO yjs_updates_log (doc_name, workspace_id, client_id, update_data, origin)
			VALUES ($1, $2, $3, $4, 'remote')
		`, [documentName, context.user.workspaceId, context.connection.clientId, update])
	},

	async onDisconnect({ documentName, context, document }) {
		await recordSessionStats(context.user.id, documentName)
	},

	extensions: [
		new Logger(),
		new Database({
			fetch: async ({ documentName }) => { /* see onLoadDocument */ },
			store: async ({ documentName, state }) => { /* see onStoreDocument */ },
		}),
	],
})

server.listen()
```

### Backpressure & Rate Limiting

```tsx
// /servers/realtime/rate-limit.ts
const opsPerClient = new Map<string, { count: number; resetAt: number }>()

export function checkOpsRateLimit(clientId: string): boolean {
	const now = Date.now()
	const entry = opsPerClient.get(clientId)
	if (!entry || now > entry.resetAt) {
		opsPerClient.set(clientId, { count: 1, resetAt: now + 1000 })
		return true
	}
	entry.count++
	return entry.count <= 50  // 50 ops/sec hard cap
}
```

### Compaction Worker (nightly)

```tsx
// /workers/yjs-compactor.ts
export async function compactDocuments() {
	const stale = await db.query(`
		SELECT doc_name FROM yjs_documents
		WHERE last_compacted_at IS NULL OR last_compacted_at < now() - INTERVAL '7 days'
		LIMIT 100
	`)
	for (const { doc_name } of stale.rows) {
		// Replay all updates into fresh doc → re-encode → store compact state
		const updates = await db.query('SELECT update_data FROM yjs_updates_log WHERE doc_name = $1 ORDER BY id', [doc_name])
		const doc = new Y.Doc()
		updates.rows.forEach(r => Y.applyUpdate(doc, r.update_data))
		const compactState = Y.encodeStateAsUpdate(doc)
		await db.query(`UPDATE yjs_documents SET state = $1, last_compacted_at = now() WHERE doc_name = $2`, [compactState, doc_name])
		// Trim old ops log entries (keep last 7 days)
		await db.query(`DELETE FROM yjs_updates_log WHERE doc_name = $1 AND occurred_at < now() - INTERVAL '7 days'`, [doc_name])
	}
}
```

### Acceptance Criteria

- 50+ concurrent collaborators على نفس document بدون lag (p99 latency < 100ms client-to-client).
- Reconnect resilience: kill server → restart → clients reconnect وthe state recovered كاملاً.
- Zero cross-workspace document access (penetration test).
- E2EE docs: server يخزن opaque bytes، فك التشفير client-side فقط.
- Compaction reduces storage by ≥ 60% بعد 7 days من active editing.
- Awareness messages dropped under backpressure but ops never lost.
