## 🛠️ Technical Implementation

### Migration 0021 — `library_items` + `highlights`

```sql
CREATE TABLE library_items (
  id TEXT PRIMARY KEY DEFAULT 'lib_' || lower(hex(random_bytes(10))),
  user_id UUID REFERENCES profiles(id),
  url TEXT,
  title TEXT,
  author TEXT,
  source_type TEXT NOT NULL, -- 'web', 'pdf', 'kindle', 'youtube'
  content_raw TEXT, -- The parsed article content (Markdown/HTML)
  reading_progress FLOAT DEFAULT 0,
  is_archived BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE highlights (
  id TEXT PRIMARY KEY DEFAULT 'hl_' || lower(hex(random_bytes(10))),
  item_id TEXT REFERENCES library_items(id) ON DELETE CASCADE,
  user_id UUID REFERENCES profiles(id),
  
  content TEXT NOT NULL,
  note TEXT,
  color TEXT DEFAULT 'yellow',
  selection_data JSONB, -- For PDF/Epub coordinates
  
  -- Spaced Repetition Data
  next_review_at TIMESTAMPTZ DEFAULT now(),
  review_count INT DEFAULT 0,
  
  created_at TIMESTAMPTZ DEFAULT now()
);
```

### The "Readwise-Killer" Engine

1.  **The Parser:** استخدام `mozilla/readability` في الـ backend لاستخراج المحتوى النظيف من أي URL.
2.  **YouTube Integration:** سحب الـ Subtitles من YouTube وتحويلها لـ readable transcript.
3.  **The Extension:** Chrome/Safari extension بتبعت الـ DOM لـ Zenith API.
4.  **Resurfacing Algorithm:** كل يوم الصبح، السيستم بيبعت Notification بـ "Your Daily Highlights" بناءً على الـ `next_review_at`.

### Zenith Aesthetics

- **Reader Mode:** خطوط Serif (Inter/Merriweather)، خلفية Sepia/Dark، عرض محدد (600px) لراحة العين.
- **Highlighter Tool:** لما تظلل نص يظهر Popup صغير للألوان والـ Notes.
