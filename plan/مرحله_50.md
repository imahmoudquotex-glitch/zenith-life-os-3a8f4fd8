### 3) Caching Pyramid

- L1: Browser cache (Cache-Control immutable)
- L2: Cloudflare CDN (60s HTML SWR, 30s JSON SWR)
- L3: Redis hot cache (5min TTL)
- L4: Postgres materialized views (refresh hourly)

```tsx
export async function cached<T>(key: string, ttl: number, fn: () => Promise<T>): Promise<T> {
  const hit = await redis.get(key)
  if (hit) return JSON.parse(hit)
  const val = await fn()
  await redis.set(key, JSON.stringify(val), 'EX', ttl)
  return val
}
```

### 10) Cost Guardrails

```tsx
export const QUOTAS = {
  ai_tokens_per_day: 200_000,
  storage_per_user_mb: 1024,
  vault_attachments_mb: 100,

  api_requests_per_minute: 60,

export async function enforceQuota(userId: string, kind: keyof typeof QUOTAS, amt: number) {
  const used = await redis.incrby(`quota:${kind}:${userId}:${windowKey(kind)}`, amt)
  if (used > QUOTAS[kind]) throw new QuotaExceeded(kind)
  if (used > QUOTAS[kind] * 0.8) await sendQuotaWarning(userId, kind, used / QUOTAS[kind])
}
```

## 2️⃣ Region Pinning (Data Residency)

```sql
-- 4150_workspace_region.sql
ALTER TABLE workspaces ADD COLUMN home_region TEXT NOT NULL DEFAULT 'us-east-1';
ALTER TABLE workspaces ADD COLUMN allowed_regions TEXT[] NOT NULL DEFAULT ARRAY['us-east-1'];
ALTER TABLE workspaces ADD CONSTRAINT chk_region
	CHECK (home_region IN ('us-east-1','eu-central-1','eu-west-2','ap-south-1','ap-northeast-1','sa-east-1'));

CREATE TABLE region_routing (
	workspace_id TEXT PRIMARY KEY,
	primary_region TEXT NOT NULL,
	replica_regions TEXT[] NOT NULL,
	updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

- المستخدم في **GDPR plan** يختار `eu-central-1` أو `eu-west-2` فقط.
- Writes تذهب دائماً للـ `primary_region`.
- Reads تذهب لأقرب replica في `allowed_regions`.
- البيانات لا تغادر `allowed_regions` (verified بـ network policies + audit).

## 3️⃣ Edge Worker (Cloudflare)

```tsx
// apps/edge/src/router.ts
export default {
	async fetch(req: Request, env: Env, ctx: ExecutionContext): Promise<Response> {
		const url = new URL(req.url)
		const country = req.cf?.country as string
		const workspace = await resolveWorkspace(req, env)
		if (workspace) {
			const targetRegion = pickReadRegion(workspace.allowed_regions, country)
			return fetch(`https://${targetRegion}.api.zenith.app${url.pathname}`, req)
		}
		// Public/static
		return env.ASSETS.fetch(req)
	},
} satisfies ExportedHandler<Env>
```
