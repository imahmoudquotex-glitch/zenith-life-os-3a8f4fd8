# مرحلة 18 — Search Re-ranking & ML Relevance (Hybrid Vector + BM25 + LTR)

<aside>
🛡️

**Canonical position:** يأتي بعد W15 (Typesense/BM25 baseline) وقبل W18 (AI Gateway). يبني طبقة ML reranking لتقديم نتائج competitive مع Notion's hybrid search.

</aside>

## 🧠 Master Architect Upgrade — Hybrid Relevance Gate

- **Tenant isolation:** كل query يمر بـ `assertWorkspaceContext()` ولا index واحد يخدم workspaces متعددة بدون namespace prefix `ws_{workspace_id}_`.
- **Embedding privacy:** vault items **ممنوع** embedding خاص بيها (ZKE — السيرفر مش شايف plaintext). فقط non-vault pages/blocks/notes تدخل vector store.
- **No PII in embeddings:** قبل embedding، DLP scanner يـ redact emails/phones/SSN/credit cards.
- **Re-ranker fairness:** LTR model يتدرب على implicit signals (clicks/dwell) لكن **بدون** demographic features. audit log كل re-ranking decision.
- **Cost guard:** embedding generation داخل `runAIWithQuota` budget. fallback لـ BM25-only لو quota exceeded.
- **Determinism:** نفس query + نفس index state → نفس النتائج (seed-based tie-breaking).

## 🧠 توسعة تنفيذية كاملة — Search Intelligence Platform

المرحلة دي لازم تتحول من "طبقة reranking" إلى **منصة بحث ذكية كاملة** تنافس Notion/Linear/Google Drive داخلياً: بحث لحظي، فهم نية المستخدم، نتائج مؤمنة بـ ACL، semantic search عربي/إنجليزي، ranking قابل للتحسين، observability، وfallback آمن لو vector أو AI وقعوا.

### Arabic/English normalization

```tsx
export function normalizeSearchText(input: string) {
  return input
    .normalize('NFKC')
    .replace(/[إأآا]/g, 'ا')
    .replace(/ى/g, 'ي')
    .replace(/ة/g, 'ه')
    .replace(/[ًٌٍَُِّْ]/g, '')
    .replace(/\s+/g, ' ')
    .trim()
}

export function detectSearchLanguage(text: string): 'ar' | 'en' | 'mixed' | 'unknown' {
  const ar = (text.match(/[\u0600-\u06FF]/g) ?? []).length
  const en = (text.match(/[a-zA-Z]/g) ?? []).length
  if (ar === 0 && en === 0) return 'unknown'
  if (ar > 0 && en > 0) return 'mixed'
  return ar > en ? 'ar' : 'en'
}
```

## 🏭 Index Worker

```tsx
export async function runSearchIndexer(workerId: string) {
  const jobs = await db.manyOrNone(`
    UPDATE search_index_outbox
    SET locked_at=now(), locked_by=$1
    WHERE id IN (
      SELECT id FROM search_index_outbox
      WHERE run_after <= now()
        AND (locked_at IS NULL OR locked_at < now() - interval '5 minutes')
      ORDER BY priority ASC, created_at ASC
      LIMIT 100
      FOR UPDATE SKIP LOCKED
    )
    RETURNING *
  `, [workerId])

  for (const job of jobs) {
    try {
      if (job.operation === 'delete') await deleteSearchDocument(job)
      else await upsertSearchDocument(job)
      await db.none(`DELETE FROM search_index_outbox WHERE id=$1`, [job.id])
    } catch (err) {
      await db.none(`
        UPDATE search_index_outbox
        SET attempts=attempts+1,
            locked_at=NULL,
            locked_by=NULL,
            last_error=$2,
            run_after=now() + (attempts + 1) * interval '2 minutes'
        WHERE id=$1
      `, [job.id, String(err).slice(0, 1000)])
    }
  }
}
```

## 🛡️ DLP Redaction قبل Embeddings

```tsx
const DLP_PATTERNS = [
  { type: 'email', rx: /[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,}/gi },
  { type: 'phone', rx: /(?:\+?\d[\d\s\-()]{7,}\d)/g },
  { type: 'credit_card', rx: /\b(?:\d[ -]*?){13,19}\b/g },
  { type: 'secret', rx: /\b(?:sk|pk|ghp|xoxb|xoxp)_[A-Za-z0-9_\-]{16,}\b/g },
  { type: 'jwt', rx: /\beyJ[A-Za-z0-9_-]+\.[A-Za-z0-9_-]+\.[A-Za-z0-9_-]+\b/g },
]

export function dlpRedactForEmbedding(text: string) {
  let redacted = text
  const hits: string[] = []
  for (const p of DLP_PATTERNS) {
    redacted = redacted.replace(p.rx, () => {
      hits.push(p.type)
      return `[REDACTED_${p.type.toUpperCase()}]`
    })
  }
  return { redacted, hits, redactionApplied: hits.length > 0 }
}
```

## ⚖️ Ranking Feature Extractor

```tsx
export type LtrFeatureVector = {
  bm25Score: number
  vectorScore: number
  rrfScore: number
  titleExactMatch: 0 | 1
  titlePrefixMatch: 0 | 1
  languageMatch: 0 | 1
  entityTypeBoost: number
  recencyDays: number
  editFrequency30d: number
  authorAffinity: number
  pathDepth: number
  clickPrior: number
}

export async function extractLtrFeatures(
  ctx: RequestContext,
  query: string,
  fused: FusedHit[],
): Promise<Array<FusedHit & { features: LtrFeatureVector }>> {
  const normalized = normalizeSearchText(query)
  const userAffinities = await loadUserAuthorAffinities(ctx.userId)
  const clickPriors = await loadClickPriors(ctx.workspaceId, normalized)

  return fused.map((hit) => {
    const title = normalizeSearchText(hit.title ?? '')
    return {
      ...hit,
      features: {
        bm25Score: hit.bm25Score ?? 0,
        vectorScore: hit.vectorScore ?? 0,
        rrfScore: hit.rrfScore,
        titleExactMatch: title === normalized ? 1 : 0,
        titlePrefixMatch: title.startsWith(normalized) ? 1 : 0,
        languageMatch: detectSearchLanguage(query) === hit.language ? 1 : 0,
        entityTypeBoost: boostEntityType(hit.entityType, normalized),
        recencyDays: daysSince(hit.sourceUpdatedAt),
        editFrequency30d: hit.editFrequency30d ?? 0,
        authorAffinity: userAffinities.get(hit.authorUserId ?? '') ?? 0,
        pathDepth: hit.path?.length ?? 0,
        clickPrior: clickPriors.get(hit.entityId) ?? 0,
      },
    }
  })
}
```

## 🧮 Scoring Fallback بدون LTR

لو مفيش model، أو فشل model loading، نستخدم scoring deterministic بسيط.

```tsx
export function fallbackScore(hit: FusedHit & { features: LtrFeatureVector }) {
  const f = hit.features
  return (
    f.rrfScore * 100 +
    f.titleExactMatch * 20 +
    f.titlePrefixMatch * 12 +
    f.languageMatch * 5 +
    f.entityTypeBoost +
    Math.max(0, 10 - Math.log1p(f.recencyDays)) +
    Math.min(5, f.authorAffinity)
  )
}
```

## 🔐 ACL Filter — Defense in Depth

```tsx
export async function aclFilter(
  hits: RankedHit[],
  ctx: RequestContext,
): Promise<RankedHit[]> {
  const allowed: RankedHit[] = []
  for (const hit of hits) {
    const canRead = await permissionResolver.canRead({
      workspaceId: ctx.workspaceId,
      userId: ctx.userId,
      entityType: hit.entityType,
      entityId: hit.entityId,
    })
    if (canRead && !hit.isVault) allowed.push(hit)
  }
  return allowed
}
```

```tsx
export type SearchBenchmarkCase = {

export function ndcgAtK(results: string[], relevant: string[], k = 10) {
  const rel = new Set(relevant)
  const dcg = results.slice(0, k).reduce((sum, id, i) => {
    const gain = rel.has(id) ? 1 : 0
    return sum + gain / Math.log2(i + 2)
  }, 0)
  const ideal = relevant.slice(0, k).reduce((sum, _, i) => sum + 1 / Math.log2(i + 2), 0)
  return ideal === 0 ? 0 : dcg / ideal
}
```

### TypeScript — Hybrid Search Pipeline

```tsx
// /src/search/hybrid.ts
import { z } from 'zod'
import { assertWorkspaceContext, assertNoVaultLeak } from '@/lib/guards'
import { runAIWithQuota } from '@/lib/ai/quota'

export const SearchQuerySchema = z.object({
	workspaceId: z.string(),
	userId: z.string(),
	query: z.string().min(1).max(500),
	filters: z.object({
		types: z.array(z.enum(['page','block','note','task'])).optional(),
		authors: z.array(z.string()).optional(),
		dateRange: z.object({ from: z.string(), to: z.string() }).optional(),
	}).optional(),
	limit: z.number().int().min(1).max(50).default(20),
})

export async function hybridSearch(input: z.infer<typeof SearchQuerySchema>) {
	assertWorkspaceContext(input.workspaceId)

	// 1) Parse + intent detection
	const intent = await detectIntent(input.query, input.workspaceId)

	// 2) Parallel: BM25 + Vector
	const [bm25Results, vectorResults] = await Promise.all([
		typesenseSearch({
			collection: `ws_${input.workspaceId}_content`,
			q: input.query,
			limit: 100,
			filters: buildTypesenseFilters(input.filters),
		}),
		vectorSearch({
			workspaceId: input.workspaceId,
			queryEmbedding: await embedQuery(input.query),
			limit: 100,
			filters: input.filters,
		}),
	])

	assertNoVaultLeak([...bm25Results, ...vectorResults])

	// 3) RRF fusion (k=60 standard)
	const fused = reciprocalRankFusion(bm25Results, vectorResults, { k: 60, topN: 50 })

	// 4) LTR re-ranking (if active model exists)
	const model = await loadActiveLtrModel(input.workspaceId)
	let reranked = fused
	if (model) {
		const features = await extractFeatures(fused, input)
		reranked = await ltrRerank(features, model)
	}

	// 5) ACL final filter (defense in depth — RLS already filters in DB layer)
	const final = await aclFilter(reranked, input.userId, input.workspaceId)

	// 6) Log for training (implicit signals collected later)
	const queryId = ulid()
	await logSearchEvent({
		queryId, workspaceId: input.workspaceId, userId: input.userId,
		query: input.query, intent, results: final.slice(0, input.limit),
	})

	return { queryId, results: final.slice(0, input.limit), intent }
}

function reciprocalRankFusion(
	bm25: SearchHit[],
	vector: SearchHit[],
	cfg: { k: number; topN: number }
) {
	const scores = new Map<string, { hit: SearchHit; score: number; bm25: number; vec: number }>()
	bm25.forEach((h, i) => {
		const s = 1 / (cfg.k + i + 1)
		scores.set(h.id, { hit: h, score: s, bm25: s, vec: 0 })
	})
	vector.forEach((h, i) => {
		const s = 1 / (cfg.k + i + 1)
		const existing = scores.get(h.id)
		if (existing) { existing.score += s; existing.vec = s }
		else scores.set(h.id, { hit: h, score: s, bm25: 0, vec: s })
	})
	return [...scores.values()]
		.sort((a, b) => b.score - a.score || a.hit.id.localeCompare(b.hit.id))  // deterministic tie-break
		.slice(0, cfg.topN)
}
```

### Embedding Worker (BGE-M3 multilingual)

```tsx
// /workers/embedding-worker.ts
import { z } from 'zod'

export const EmbedJobSchema = z.object({
	entityType: z.enum(['page','block','note','task']),
	entityId: z.string(),
	workspaceId: z.string(),
	content: z.string().max(8192),
	isVault: z.boolean(),  // hard-fail if true
})

export async function processEmbedJob(job: z.infer<typeof EmbedJobSchema>) {
	if (job.isVault) throw new Error('FATAL: vault content must never be embedded')

	// DLP redaction before embedding
	const { redacted, hasRedactions } = await dlpRedact(job.content)

	// Skip empty after redaction
	if (redacted.trim().length < 10) return { skipped: true }

	const embedding = await callBgeM3(redacted)  // 1024-dim
	const contentHash = sha256(job.content)

	await db.query(`
		INSERT INTO search_embeddings (id, workspace_id, entity_type, entity_id, content_hash, embedding, model_version, redaction_applied)
		VALUES ($1,$2,$3,$4,$5,$6,'bge-m3-v1.5',$7)
		ON CONFLICT (id) DO UPDATE SET
			embedding = EXCLUDED.embedding,
			content_hash = EXCLUDED.content_hash,
			updated_at = now()
	`, [ulid(), job.workspaceId, job.entityType, job.entityId, contentHash, toPgVector(embedding), hasRedactions])
}
```

### LTR Training Pipeline (offline, nightly)

```tsx
// /workers/ltr-trainer.ts
import xgb from 'xgboost-node'

export async function trainLtrModel(workspaceId: string | null) {
	const training = await db.query(`
		SELECT query_id, query_text, result_id, result_position,
		       clicked, dwell_seconds, bm25_score, vector_score, rrf_score
		FROM search_click_log
		WHERE ${workspaceId ? 'workspace_id = $1' : 'true'}
		  AND occurred_at > now() - INTERVAL '90 days'
	`, workspaceId ? [workspaceId] : [])

	const features = training.rows.map(r => ({
		query_id: r.query_id,
		label: r.clicked ? (r.dwell_seconds > 30 ? 2 : 1) : 0,  // graded relevance
		features: [
			r.bm25_score, r.vector_score, r.rrf_score,
			r.result_position,
			// + entity recency, edit frequency, author proximity, etc.
		],
	}))

	const model = await xgb.train({
		objective: 'rank:ndcg',
		eval_metric: 'ndcg@10',
		max_depth: 6,
		eta: 0.1,
		num_round: 100,
	}, features)

	const artifactUrl = await uploadToR2(`ltr/${workspaceId ?? 'global'}/${Date.now()}.json`, model.serialize())
	await db.query(`
		INSERT INTO ltr_models (id, workspace_id, model_artifact_url, feature_schema_version, trained_at, ndcg_at_10, mrr, is_active)
		VALUES ($1,$2,$3,1,now(),$4,$5,true)
	`, [ulid(), workspaceId, artifactUrl, model.evals.ndcg_at_10, model.evals.mrr])

	// Atomic activation
	await db.query(`
		UPDATE ltr_models SET is_active = false
		WHERE workspace_id IS NOT DISTINCT FROM $1 AND id != $2
	`, [workspaceId, model.id])
}
```
