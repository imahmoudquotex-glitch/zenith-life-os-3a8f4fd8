# 🔥 مرحلة 32 — Local-First + E2EE Sync + Offline Mode

## 🧠 توسعة تنفيذية كاملة — Local-First Privacy Operating System

مرحلة 32 لازم تكون واحدة من أقوى مراحل الخطة لأنها تنقل المنتج من “ويب أب سريع” إلى **منصة ملكية بيانات حقيقية**: كل شيء يعمل محليًا أولًا، البحث محلي، الكتابة بدون إنترنت، المزامنة ذكية، التشفير اختياري لكن قوي، والسيرفر لا يكون مصدر الحقيقة الوحيد. الهدف هو منافسة Notion في سهولة الاستخدام، ومنافسة Obsidian/Anytype/Standard Notes في الخصوصية والـ offline.

<aside>
🔒

تعليمات للـ AI/Claude Code: لا تبدأ من UI. ابدأ من Local Data Model → Operation Log → Sync Protocol → Encryption Envelopes → Conflict/CRDT Merge → Background Workers → Recovery/Export. أي feature لا تعمل offline تعتبر غير مكتملة.

</aside>

### الهدف النهائي للمرحلة

- التطبيق يفتح ويقرأ ويكتب ويبحث **بدون إنترنت بالكامل**.
- كل جهاز له `device_id` ومفتاح عام، ويشارك في sync protocol واضح.
- كل تعديل يتحول إلى operation محلية ثم sync لاحقًا.
- E2EE workspace يخزن على السيرفر ciphertext فقط.
- local search يستخدم SQLite FTS5، وليس search server.
- conflict handling يعتمد CRDT/Yjs أولًا، وواجهة user-facing عند الحالات النادرة.
- export كامل: Markdown + JSON + SQLite dump + encrypted backup.
- recovery kit واضح لاسترجاع workspace مشفر.
- metrics تفرق بين sync failures وencryption failures وoffline queue backlog.

## 🧭 Mental Model للـ AI

```
User Edit
  ↓
Local SQLite Transaction
  ↓
Yjs / Operation Log
  ↓
Local FTS Index Update
  ↓
Dirty Queue
  ↓
Encryption Layer (if E2EE)
  ↓
Sync Push / Pull
  ↓
Remote Encrypted Blob Store
  ↓
Ack Clock + Local Cleanup
```

### قواعد صارمة

- ممنوع blocking UI أثناء sync.
- ممنوع فقدان تعديل محلي حتى لو التطبيق اتقفل.
- ممنوع تخزين مفاتيح E2EE في localStorage.
- ممنوع إرسال plaintext من E2EE workspace للسيرفر أو AI أو analytics.
- ممنوع الاعتماد على server search في encrypted workspaces.
- ممنوع اعتبار “last write wins” حل conflict عام.
- ممنوع sync بدون idempotency key وdevice clock.

## 💾 Local Write Path

```tsx
export async function applyLocalEdit(args: {
  workspaceId: string
  documentId: string
  update: Uint8Array
  title?: string
}) {
  return localDb.transaction(async (tx) => {
    const clock = await nextLocalClock(tx, args.workspaceId, args.documentId)
    const doc = await tx.get<LocalDocumentRecord>(
      `SELECT * FROM local_documents WHERE id=?`,
      [args.documentId],
    )

    const merged = mergeYjsState(doc?.yjsState, args.update)
    await tx.run(`
      INSERT INTO local_documents(id, workspace_id, entity_type, title, yjs_state, local_clock, dirty, created_at, updated_at)
      VALUES(?,?,?,?,?,?,1,?,?)
      ON CONFLICT(id) DO UPDATE SET
        yjs_state=excluded.yjs_state,
        local_clock=excluded.local_clock,
        dirty=1,
        title=coalesce(excluded.title, local_documents.title),
        updated_at=excluded.updated_at
    `, [args.documentId, args.workspaceId, 'page', args.title ?? null, merged, clock, Date.now(), Date.now()])

    await tx.run(`
      INSERT INTO local_ops(id, workspace_id, document_id, device_id, local_clock, op_kind, yjs_update, idempotency_key, created_at)
      VALUES(?,?,?,?,?,?,?,?,?)
    `, [ulid(), args.workspaceId, args.documentId, await getDeviceId(), clock, 'yjs_update', args.update, `${args.documentId}:${clock}`, Date.now()])

    await updateLocalFts(tx, args.documentId, merged)
    return { clock }
  })
}
```

## 🔄 Sync Protocol

```tsx
export class LocalFirstSyncEngine {
  private running = false

  async tick() {
    if (this.running) return
    this.running = true
    try {
      await this.pushPendingOps()
      await this.pullRemoteOps()
      await this.compactLocalIfNeeded()
    } finally {
      this.running = false
    }
  }

  async pushPendingOps() {
    const ops = await localDb.all(`
      SELECT * FROM local_ops
      WHERE sync_status IN ('pending','failed')
      ORDER BY created_at ASC
      LIMIT 100
    `)
    for (const op of ops) {
      await localDb.run(`UPDATE local_ops SET sync_status='pushing' WHERE id=?`, [op.id])
      try {
        const payload = await maybeEncryptOp(op)
        await api.post('/api/v1/sync/push', payload, {
          headers: { 'Idempotency-Key': op.idempotency_key },
        })
        await localDb.run(`UPDATE local_ops SET sync_status='acked', acked_at=? WHERE id=?`, [Date.now(), op.id])
      } catch (err) {
        await localDb.run(`UPDATE local_ops SET sync_status='failed', error_code=? WHERE id=?`, [normalizeSyncError(err), op.id])
      }
    }
  }

  async pullRemoteOps() {
    const checkpoints = await localDb.all(`SELECT * FROM sync_checkpoints_local`)
    for (const cp of checkpoints) {
      const remote = await api.get(`/api/v1/sync/pull?documentId=${cp.document_id}&after=${cp.last_remote_clock}`)
      for (const op of remote.ops) {
        const update = await maybeDecryptRemoteOp(op)
        await applyRemoteYjsUpdate(cp.document_id, update, op.clock)
      }
    }
  }
}
```

## 🔐 E2EE Key Flow

```tsx
export async function setupE2eeWorkspace(args: {
  workspaceId: string
  userPassword: string
}) {
  await sodium.ready
  const workspaceKey = sodium.randombytes_buf(32)
  const deviceKeys = await deviceKeyStore.getOrCreate()
  const envelope = sodium.crypto_box_seal(workspaceKey, deviceKeys.publicKey)
  const recoveryKit = await createRecoveryKit(workspaceKey)

  await api.post('/api/v1/encryption/key-envelope', {
    workspaceId: args.workspaceId,
    deviceId: deviceKeys.deviceId,
    publicKey: bytesToBase64(deviceKeys.publicKey),
    encryptedWorkspaceKey: bytesToBase64(envelope),
    recoveryKitHash: await hashRecoveryKit(recoveryKit.words),
  })

  await secureKeyStore.saveWorkspaceKey(args.workspaceId, workspaceKey)
  return { recoveryKit }
}
```

### مكان حفظ المفاتيح

- Desktop PWA: WebCrypto non-extractable keys + IndexedDB encrypted envelope.
- Electron لاحقًا: OS Keychain.
- iOS/Android: Keychain/Keystore.
- ممنوع `localStorage`.
- recovery kit يظهر مرة واحدة مع confirmation.

## 🧯 Recovery + Backup

```tsx
export async function createEncryptedBackup(args: {
  workspaceId: string
  destination: 'file' | 'drive'
}) {
  const dump = await exportSqliteDump(args.workspaceId)
  const key = await secureKeyStore.getWorkspaceKey(args.workspaceId)
  const encrypted = await encryptBlob(dump, key)
  const checksum = await sha256(encrypted.ciphertext)
  return writeBackupFile({
    filename: `workspace-${args.workspaceId}-${Date.now()}.zenithbackup`,
    ciphertext: encrypted.ciphertext,
    nonce: encrypted.nonce,
    checksum,
  })
}
```

### Recovery requirements

- restore from recovery kit.
- restore from encrypted backup.
- device revocation stops future key envelope access.
- lost recovery kit = server cannot recover E2EE workspace.
- clear user-facing warning before enabling E2EE.

## 🛡️ Conflict Strategy

```tsx
export async function applyRemoteYjsUpdate(documentId: string, update: Uint8Array, remoteClock: number) {
  await localDb.transaction(async (tx) => {
    const doc = await loadLocalYDoc(tx, documentId)
    const beforeVector = Y.encodeStateVector(doc)
    Y.applyUpdate(doc, update)
    const afterVector = Y.encodeStateVector(doc)

    await saveLocalYDoc(tx, documentId, doc, { remoteClock })
    if (stateVectorChangedUnexpectedly(beforeVector, afterVector)) {
      await conflictLog.maybeRecordAutoMerge(documentId, beforeVector, afterVector)
    }
  })
}
```

### Conflict UI يظهر فقط عند:

- حذف صفحة على جهاز وتعديلها على جهاز آخر.
- تغيير metadata non-CRDT بنفس الوقت.

## 🔐 E2EE Flow

```tsx
// lib/crypto/e2ee.ts
import sodium from 'libsodium-wrappers'

export async function encryptDocument(yDocState: Uint8Array, workspaceKey: Uint8Array) {
  const nonce = sodium.randombytes_buf(24)
  const ciphertext = sodium.crypto_secretbox_easy(yDocState, nonce, workspaceKey)
  return { ciphertext, nonce }
}

export async function shareKeyToDevice(workspaceKey: Uint8Array, devicePublicKey: Uint8Array) {
  // sealed box: only the device with matching private key can open
  return sodium.crypto_box_seal(workspaceKey, devicePublicKey)
}

export async function generateRecoveryKit(workspaceKey: Uint8Array) {
  // BIP39 mnemonic encoding للـ user to write down
  return bip39.entropyToMnemonic(workspaceKey)
}
```

## 🔄 Sync Engine

```tsx
// lib/sync/engine.ts
export class SyncEngine {
  async pushChanges() {
    const dirty = await localDb.all('SELECT * FROM documents WHERE is_dirty = 1')
    for (const doc of dirty) {
      const update = Y.encodeStateAsUpdate(doc.yjs)
      const encrypted = await encryptDocument(update, this.workspaceKey)
      await api.post('/sync/push', { documentId: doc.id, ...encrypted })
      await localDb.run('UPDATE documents SET is_dirty=0, synced_at=? WHERE id=?', Date.now(), doc.id)
    }
  }

  async pullChanges() {
    const remote = await api.get('/sync/pull', { since: this.lastSyncClock })
    for (const change of remote) {
      const update = await decryptDocument(change.ciphertext, change.nonce, this.workspaceKey)
      const yDoc = this.getOrLoadDoc(change.documentId)
      Y.applyUpdate(yDoc, update) // CRDT merge = no conflicts
    }
  }

  startBackgroundSync() {
    setInterval(() => this.pushChanges(), 5000)
    this.ws.on('change', () => this.pullChanges())
  }
}
```
