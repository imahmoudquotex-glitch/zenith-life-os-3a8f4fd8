## A. Cross-Wave Invariants (تسري بدون استثناء)

- **Desktop-only PWA:** app shell offline precached (shell.html/css/js/fonts/icons)، min window 1024×640 / optimal 1440×900، installable manifest كامل (display=standalone, theme_color=#0a0a0a, background_color=#0a0a0a, icons 192/512/maskable)، keyboard-first (Command Palette ⌘K، Quick Switcher ⌘P، Smart Search ⌘/)، File System Access API للـ import المباشر، Web Push عبر VAPID + endpoint-based subscriptions.
- **Dark Mode invariant:** tokens ثابتة (`--bg-canvas:#0a0a0a`, `--bg-surface:#131313`, `--bg-elevated:#1c1c1c`, `--text-primary:#fafafa`, `--text-secondary:#a3a3a3`, `--accent-neon:#4ade80`, `--accent-danger:#ef4444`). CI visual snapshot يرفض أي surface فاتح. WCAG 2.2 AA: contrast ≥ 4.5:1. `prefers-reduced-motion` يُحترم (animations max 200ms).
- **Multi-tenant fortress:** كل tenant table = `workspace_id TEXT NOT NULL` + FK + RLS + FORCE RLS + `<table>_isolation` policy + pgTAP cross-tenant test. `current_workspace_id()` و`current_user_id()` set في كل request. ESLint يمنع cross-workspace JOIN بدون audit. Tenant context middleware يفشل request لو context مش set.
- **Zero-Knowledge Encryption (ZKE):** نستخدم ZKE للخزنة (ZKP يُحفظ لـ contexts بـ بروتول proof حقيقي مثل zk-SNARK/zk-STARK).
    - KDF: Argon2id (t=3, m=64MiB, p=4, salt=16B random per user).
    - Symmetric: XChaCha20-Poly1305 AEAD، nonce 24B random، AAD = `{workspace_id, item_id, version}`.
    - Asymmetric: X25519 ECDH + sealed boxes (libsodium) للـ sharing.
    - Master Key (MK) = Argon2id(passphrase, salt) → in-memory only في WASM locked module → wiped on tab close.
    - Item Encryption Key (IEK) random per item، encrypted by MK، stored as `wrapped_iek BYTEA`.
    - ممنوع plaintext في: AI، analytics، audit، IndexedDB، localStorage، SSR cache، Sentry events، logs.
    - Recovery: Shamir's Secret Sharing 3-of-5 opt-in.
    - HW unlock: WebAuthn `largeBlob` extension.
    - Audit: `{action, item_id_hash, ts, ip, ua}` فقط، بدون plaintext.
- **AI safety perimeter:** `runAIWithQuota({sensitivity, workspaceId, userId, prompt, schema, ...})` entrypoint وحيد. Sensitivity levels: `none|low|medium|high`. `high` = AI ممنوع كلياً (vault, journals, moods, financial PII, biometric). Context scrubbing قبل prompt (regex + PII model). Zod structured output. System prompt مع `<user_input>...</user_input>` containment ضد prompt injection. Atomic reserve/complete/refund عبر RPCs. ESLint يمنع `openai`/`@anthropic-ai/sdk` خارج `packages/ai/src/providers/`. Whisper Mode global toggle يعطل AI كلياً للـ user. Cost ledger مفصل في `ai_usage_events`.
- **Observability:** OpenTelemetry auto-instrument (http, pg, redis, fetch)، root span `<method> <route>` مع `app.workspace_id`, `app.user_id`, `app.request_id`, `app.trace_id`. pino JSON logs. Prometheus metrics مع SLOs: API p99<300ms، DB p99<50ms، AI p95<5s، error rate<0.1%. PagerDuty/Slack alerts على SLO breach + DLQ depth + error budget exhaustion.
- **Disaster Recovery:** RTO≤1h، RPO≤15min، PITR 7d hot/30d warm/1y cold (encrypted S3 Glacier)، quarterly restore drills موثقة، multi-region active-passive (eu-west-1 primary, us-east-1 standby) من Wave 31، tenant data export bundle مشفر متاح دائماً.
- **Accessibility WCAG 2.2 AA:** semantic HTML، keyboard nav كاملة، focus visible، ARIA labels، screen reader tested (NVDA + VoiceOver)، `prefers-reduced-motion` + `prefers-contrast` يحترموا.
- **Donations-only economics:** كل المميزات مجانية للجميع. donations اختيارية بحت بدون أي feature gate. badge احتفالي فقط بدون صلاحيات. جدول `donations(id, user_id, amount_cents BIGINT, currency CHAR(3), provider, provider_tx_id, status, anonymous BOOLEAN, created_at)`.
- **Supply chain hardening:** SBOM (CycloneDX) في كل release، OSSF Scorecard ≥ 7.0، Sigstore signing للـ artifacts، Dependabot + Renovate، gitleaks + Semgrep + OWASP ZAP في CI.

## B. Engineering Contracts

- **Testing matrix:** Vitest unit ≥85% (services, repos)، pgTAP 100% RLS tables، Playwright E2E critical paths، Chromatic visual regression، k6 load testing، Semgrep+ZAP+gitleaks (0 high/critical).
- **Migration header إجباري:** `-- File: NNNN__<slug>.sql / Wave / Description / Author / Created / Idempotent:YES / Rollback:forward-fix only` + BEGIN/COMMIT + IF NOT EXISTS + concurrent index creation للـ production.
- **Error registry:** كل error usage لازم registry entry في `ERROR_CODES`. CI يرفض missing entries.
- **Encryption primitives forbidden:** AES-ECB، MD5، SHA1، 3DES، RC4، weak DH groups. CI `check-encryption-primitives.ts` يفرض ذلك.
- **Performance budgets:** TTI<2s (cached) / <4s (cold)، FCP<1.2s، main chunk<200KB gzipped، vendor<300KB، DB query p95<50ms، API p95<200ms، AI p50<2s.
