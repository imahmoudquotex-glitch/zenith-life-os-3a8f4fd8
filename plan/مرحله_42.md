# المرحلة 39 — محرك الإدمان (LifeOS Engagement Engine)

## 🧠 توسعة ضخمة — Humane Engagement & Habit Formation Operating System

محرك “الإدمان” هنا لازم يتنفذ كـ **محرك ارتباط صحي Humane Engagement Engine**: يخلي المستخدم يرجع كل يوم لأنه شايف قيمة ووضوح وتقدم، مش لأنه مضغوط أو خايف يخسر حاجة. الهدف هو بناء Ritual Loop ذكي: يبدأ اليوم بتوجيه واضح، يحافظ على الاستمرارية، يحتفل بالإنجاز، يساعد على الرجوع بعد الانقطاع، ويمنع أي Dark Patterns أو مقارنات اجتماعية.

<aside>
🧠

القاعدة التنفيذية: كل ميزة Engagement لازم تكون قابلة للإيقاف، خاصة افتراضيًا، داعمة نفسيًا، ولا تستخدم الخوف أو الذنب أو الندرة المصطنعة.

</aside>

### المميزات الجديدة المطلوبة داخل المحرك

- **Daily Ritual Hub:** شاشة يومية واحدة تجمع Morning Brief، أهم 3 مهام، العادات، التركيز، والمراجعة المسائية.
- **Adaptive Habit Loops:** النظام يتعلم أفضل وقت لكل عادة بدل إرسال تذكيرات عشوائية.
- **Comeback Mode:** لو المستخدم انقطع، يرجعه بخطة لطيفة “ابدأ صغير” بدل كسر نفسي.
- **Streak Pause / Vacation Mode:** إيقاف الاستمرارية أثناء السفر/المرض/الضغط بدون عقوبة.
- **Streak Freeze Tokens:** حماية الاستمرارية من أيام الطوارئ، مع حدود صحية وواضحة.
- **Life Tree / Garden:** visualization طويل المدى بدل مجرد أرقام.
- **Focus Momentum:** حلقات إنجاز للـ deep work بدون تشجيع العمل المفرط.
- **Reward Catalog:** themes، stickers، sounds، profile titles، badges؛ كلها cosmetics فقط.
- **Reflection Memory:** AI يستخرج أنماط إيجابية من reflections بدون ضغط.
- **Weekly Reset Ritual:** بداية أسبوع بتخطيط واقعي لا يحمل المستخدم فوق طاقته.
- **Yearly Wrapped:** ملخص سنوي خاص، قابل للمشاركة فقط بموافقة صريحة.
- **Wellbeing Dashboard:** يوضح هل التفاعل صحي أم زائد عن اللزوم.

## 🔥 Streak Engine إنساني

```tsx
export async function updateStreakAfterEvent(event: EngagementEvent) {
  const prefs = await engagementPrefsRepo.getOrDefault(event.workspaceId, event.userId)
  if (!prefs.streaksVisible) return { skipped: true, reason: 'streaks_hidden' as const }

  const streakKind = mapEventToStreak(event.kind)
  if (!streakKind) return { skipped: true, reason: 'no_streak_kind' as const }

  const pause = await streakPauseRepo.activeFor(event.userId, streakKind, event.localDate)
  if (pause) return { paused: true, streakProtected: true }

  const current = await streakRepo.getOrCreate(event.userId, streakKind)
  const next = computeNextStreak(current, event.localDate)

  await streakRepo.save(next)

  if (next.milestoneReached) {
    await rewardService.maybeGrantMilestoneReward({
      workspaceId: event.workspaceId,
      userId: event.userId,
      streakKind,
      streakLength: next.current,
    })
  }

  return next
}
```

## 🎲 Variable Reward لكن أخلاقي

```tsx
export async function maybeGrantVariableReward(event: EngagementEvent) {
  const prefs = await engagementPrefsRepo.getOrDefault(event.workspaceId, event.userId)
  if (!prefs.variableRewardsEnabled) return null

  const recentRewards = await rewardRepo.countLastDays(event.userId, 7)
  if (recentRewards >= 3) return null // يمنع الإفراط

  const probability = await rewardProbability(event)
  if (secureRandomFloat() > probability) return null

  const reward = pickEthicalReward(event)
  await grantRewardOnce({
    workspaceId: event.workspaceId,
    userId: event.userId,
    rewardCode: reward.code,
    sourceKind: event.kind,
    sourceId: event.id,
  })

  return reward
}
```

## 🧘 Wellbeing Controls

```tsx
export async function evaluateWellbeingSignals(ctx: RequestContext) {
  const signals: WellbeingSignal[] = []
  const lateUsage = await usageRepo.minutesAfter(ctx.userId, '23:00')
  const notifications = await notificationRepo.countToday(ctx.userId)
  const focusMinutes = await focusRepo.minutesToday(ctx.userId)

  if (lateUsage > 45) signals.push('late_night_usage')
  if (notifications > 6) signals.push('too_many_notifications')
  if (focusMinutes > 360) signals.push('overwork_risk')

  for (const signal of signals) {
    await wellbeingSignalRepo.record(ctx, signal)
    await applyWellbeingAdjustment(ctx, signal)
  }
}

async function applyWellbeingAdjustment(ctx: RequestContext, signal: WellbeingSignal) {
  if (signal === 'too_many_notifications') {
    await engagementPrefsRepo.temporarilyReduceNotifications(ctx.userId)
  }
  if (signal === 'late_night_usage') {
    await notificationService.suggestQuietHours(ctx.userId)
  }
}
```

## 🧠 Master Architect Upgrade — Humane Engagement Gate

- **Rename internally:** اعتبرها Engagement Engine لا "إدمان" في الكود والرسائل العامة. الهدف ritual صحي لا compulsive use.
- **No manipulation:** variable rewards لازم تكون opt-in/disable-able، respect reduced motion، بدون guilt/FOMO/scarcity أو loss aversion قاسي.
- **Private by default:** streaks/rewards/wrapped private، sharing explicit، ولا global leaderboard أو مقارنة مع آخرين.
- **AI insight guard:** كل AI insight يمر `assertHumane` + DLP + no-vault؛ لا يستخدم الصحة/المزاج للضغط على المستخدم.
- **Wellbeing controls:** quiet hours، pause streaks، hide XP، disable celebrations، وexport/delete engagement history.

المرحلة دي هي اللي بتخلي المستخدم يفتح LifeOS كل صبح أوتوماتيك. مفيش social comparison ولا leaderboards ضد ناس تانية — كل شيء **أنت ضد نفسك**. بنبني Daily Ritual (Morning Brief + Evening Reflection)، Streak system متعدد الطبقات (مهام، عادات، journal، meditation)، **Streak Freeze Tokens** عشان مكسرش اللي بنيته، Variable Ratio Rewards (مكافآت عشوائية على فترات غير منتظمة — اللي بيخلي السلوك الأقوى إدماناً علمياً)، Visual Progress يكبر معاك (شجرة بتنمو، حلقات بتتلون، XP rings)، Daily Insight من الـ AI بيتأقلم مع لغتك، Push Notifications ذكية في الوقت المناسب لكل واحد (مش spam). كل ده **free forever** ومش بيستغل المستخدم بطريقة سامة — Anti-Dark-Pattern: مفيش guilt trips ولا FOMO فارغ، الكل بـ celebration و رفع للمعنويات.

### 2) Streak engine متعدد المسارات

```tsx
// src/lib/engagement/streak.ts
export async function bumpStreak(userId: string, kind: StreakKind, today = todayISO()) {
  const s = await db.streaks.where({ user_id: userId, kind }).one() ?? await initStreak(userId, kind)
  const last = s.last_date ? new Date(s.last_date) : null
  const yest = addDays(new Date(today), -1).toISOString().slice(0,10)
  // اليوم نفسه؟ مفيش تغيير
  if (s.last_date === today) return s
  // امبارح؟ continue
  if (s.last_date === yest) {
    const current = s.current + 1
    await db.streaks.update({ current, longest: Math.max(s.longest, current), last_date: today }).where({ user_id: userId, kind })
    await maybeGrantReward(userId, kind, current)
    await growLifeTree(userId)
    return { ...s, current, last_date: today }
  }
  // مكسور؟ شوف فيه freeze
  if (s.frozen_until && s.frozen_until >= yest) {
    return s   // الـ freeze حماه
  }
  // اتكسر فعلاً
  await db.streaks.update({ current: 1, last_date: today }).where({ user_id: userId, kind })
  await dropTreeHealth(userId, 5)
  return { ...s, current: 1, last_date: today }
}

export async function useFreezeToken(userId: string, kind: StreakKind, day: string) {
  const used = await db.streak_freezes.where({ user_id: userId, kind }).whereRaw("used_on >= date_trunc('month', now())").count()
  if (used >= 2) return { ok: false, reason: 'MONTHLY_LIMIT' }   // 2 شهرياً free
  await db.streak_freezes.insert({ user_id: userId, kind, used_on: day, source: 'monthly_free' })
  await db.streaks.update({ frozen_until: day }).where({ user_id: userId, kind })
  return { ok: true }
}
```

### 3) Variable Ratio Reward scheduler

```tsx
// src/lib/engagement/rewards.ts
// متغير بمتوسط 1 reward لكل ~7 إجراءات إيجابية، ±3
const LAMBDA = 1/7

export async function maybeGrantReward(userId: string, trigger: string, streakLen: number) {
  // مع streak أطول، احتمال أعلى
  const p = LAMBDA * (1 + Math.log10(streakLen + 1) * 0.3)
  if (Math.random() > p) return null
  const tier = pickRarity(streakLen)
  const reward = pickFromCatalog(tier)
  await db.engagement_rewards.insert({ user_id: userId, ...reward })
  return reward
}
function pickRarity(len: number) {
  const r = Math.random()
  if (len > 100 && r < 0.05) return 'legendary'
  if (len > 30  && r < 0.15) return 'epic'
  if (r < 0.30) return 'rare'
  return 'common'
}
```

```tsx
export async function sendSmart(userId: string, payload: PushPayload) {
  const ns = await db.notification_schedule.where({ user_id: userId }).one()

  return sendPush(userId, payload)
```
