# @zenith/vault

Vault client: encrypt/decrypt vault items (ZKE)

## Installation

This package is part of the Zenith Life OS monorepo.

```bash
# From workspace root:
pnpm install
```

## Usage

```typescript
import { encryptVaultItem, decryptVaultItem } from '@zenith/vault'
```
